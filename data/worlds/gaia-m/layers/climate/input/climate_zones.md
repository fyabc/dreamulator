# 气候带

### 自变量

无新增（综合以上各节）。

### 因变量 — 主要气候带

| 纬度范围 | 气候类型 | 核心特征 |
|----------|---------|---------|
| 0°–30° | 赤道超级雨林 / 双峰型超级季风 | 每年两次直射，两个雨季+两个旱季；裂谷海放大水汽深入腹地 |
| 30°–55° | 副热带超级荒漠（季节性） | 哈德里环流下沉支；冬季极旱，夏季 ITCZ 边缘扫过带来短暂暴雨 |
| 55°–81° | 温带/亚寒带海洋性气候 | 极锋推至 60°；大陆西岸终年温和湿润，内陆过渡为亚寒带针叶林 |
| >81° | 极地苔原 + 高山冰帽 | 仅海拔 >1500 m 处有永久冰盖（红矮星近红外冰雪反照率低，仅靠高程维持） |

### 因变量 — 特殊小气候区

| 类型 | 位置 | 机制 |
|------|------|------|
| 巨型季风区 | Aegis 大陆中部裂谷海 | 裂谷海季节性热低压 → 超级季风，干湿季分明 |
| 超级潮汐盐沼带 | 平缓大陆架海岸 | 78 m 潮差 → 45 km 宽干湿交替盐沼泥炭沼泽 |
| 地热绿洲 | 纬度 >70° 的火山热点周围 50 km | 地热流维持地表 >20°C，温泉密布的温带绿洲 |
| 雨影极端荒漠 | Anti-Aegis 大陆西海岸山脉背风坡 | 类似阿塔卡马沙漠，可能数百年无降水 |

### 因变量 — 新奇气候现象

| 现象 | 机制 |
|------|------|
| **经度气候不对称** | 巨行星 Aegis 固定在次行星半球天空 → 红外辐射+反射光使该半球夜温高 1.5–2.0°C → Aegis 大陆整体比同纬度 Anti-Aegis 大陆更湿润 |
| **无台风带** | 纬度 <15° 科氏力不足以形成闭合气旋 |
| **行星尺度飑线** | 纬度 ~20° 可形成绵延上万公里、持续数周的超级飑线（非紧凑型台风） |
| **快闪生态节律** | 20.1 天季节太快 → 植物以温湿度触发开花（3–5 天），动物无冬眠仅有短期蛰伏或快速经向迁徙 |

### 北极永久冰盖维持条件

| 条件 | 当前状态 |
|------|---------|
| 动力学孤立 | ✅ 北极孤立大陆 + 极地环极洋流 ACC |
| 地形高程 | ✅ 海拔 >1500 m 高原区（冰盖仅存于此） |
| 极夜冷却 | ✅ 31 天极夜 → 中心气温 <−40°C |
| 冰盖脆弱性 | ⚠️ 轨道偏心率回升或 ACC 中断 → 几千年内崩塌 |
