# 卫星 Gaia-M — 轨道动力学

### 自变量

| 参数 | 值 | 备注 |
|------|-----|------|
| 卫星轨道倾角 i | 9.0° | 卫星轨道面与黄道面夹角 |
| 轨道偏心率 e | **0.0025** | 由 1:2:4 轨道共振泵浦维持 |
| 1:2:4 轨道共振 | 三颗巨行星构成拉普拉斯共振 | 防止巨行星坠入主星，维持偏心率不衰减 |

### 因变量

| 参数 | 值 | 推导依据 |
|------|-----|---------|
| 黄赤交角 | 9.0° | 卫星被巨行星潮汐锁定 → 自转轴垂直于轨道面 |
| 回归线 | 南北纬 9° | |
| 极圈（极昼极夜界限） | 南北纬 81° | |
| 季节周期 | **80.5 天** | 等于巨行星公转周期；每季约 **20.1 天** |
| 最大垂直偏移 | 115,722 km | 倾角导致卫星在轨道最高点距黄道面的垂直距离 |
| 日食遮蔽率 | 2.1% | 全年 80.5 天中约 58 天无日食 |
| 日全食季 | 每年 2 次，每次 11.9 天 | 每 78h 穿过本影一次，约 3–4 次日全食（每次~2.5h） |
| 日偏食季 | 每年约 2 次 | |
| 外侧行星 2 轨道 | **0.444 AU** | 共振 2:1 → 0.2795 × 2^(2/3) |
| 外侧行星 3 轨道 | **0.704 AU** | 共振 4:1 → 0.2795 × 4^(2/3) |
| 卫星轨道外迁速率 | ~1.7 m/年 | 角动量正向传递（巨行星自转快于卫星公转） |
