# 太空时代发展推演

## 核心轨道约束

| 参数 | 值 | 推导依据 |
|------|-----|---------|
| 逃逸速度 v_e | **11.85 km/s** | √(2GM_s/R_s)；地球 11.2 km/s |
| 第一宇宙速度 v₁ | **8.38 km/s** | √(GM_s/R_s)；地球 7.9 km/s |
| 同步轨道半径 r_sync | **98,488 km**（高度 91,671 km） | (GM_sT_rot²/4π²)^(1/3) |
| Gaia-M 希尔半径 R_H_s | **68,220 km**（10.0 R_s） | a_m × (M_s/3M_p)^(1/3) |
| **GEO 存在性** | **❌ 不存在** | r_sync (98,488 km) > R_H_s (68,220 km)；任何超出希尔边界的卫星被 Aegis 潮汐力剥夺 |
| r_sync / R_H_s | **1.44** | 同步轨道在希尔球外 44% |

## 科技树分歧总表

| 发展维度 | 地球 | Gaia-M | 核心物理根源 |
|---------|------|--------|-------------|
| **通信卫星架构** | GEO 单星覆盖全球 | **大椭圆轨道 (Molniya) / 低轨巨型星座** | GEO 物理不存在（r_sync > R_H_s） |
| **早期航天动力** | 化学火箭主导 | **核热推进 (NTR) 极早商用** | 1.05g 重力井 + 11.85 km/s 逃逸速度 + 强辐射带要求大防辐射屏蔽质量 |
| **抗辐射电子学** | 硅基芯片 + 局部加固 | **碳化硅 (SiC) 基抗辐射芯片 / 光学计算** | 狂暴范艾伦辐射带导致硅芯片频繁单粒子翻转 |
| **巨行星探测** | 飞掠为主 | **磁帆 (Magsail) 悬停与无工质减速** | 距离极近（74 万 km，通信延迟 ~2.5s）+ Aegis 强磁场 → 超导线圈与磁层等离子体相互作用实现无工质减速 |
| **内行星探测** | 太阳帆 / 气动减速 | **红外帆 + Aegis 逆向引力弹弓** | 红矮星光度弱（NIR 峰值）但距离近（0.15–0.2 AU）；去内行星需大幅减速 |
| **外行星探测** | 几十年一遇的引力弹弓窗口 | **常态化共振链"引力弹弓列车"** | 1:2:4 拉普拉斯共振链 → 极规律的低能量转移窗口；多目标飞掠成为常态 |
| **深空探索动机** | 寻找新家园 / 防范撞击 | **纯粹资源开采与基础物理验证** | 红矮星万亿年寿命 → 无生存焦虑，文明心态极度从容 |
| **航空技术** | 高机动战斗机 | **平流层巨型飞艇 / 太阳能滑翔机** | 弱科氏力 → 全球无台风 → 大气极其稳定；弥补 GEO 缺失的通信/气象中继需求 |

## 航空与低轨时代

- **航空**：慢自转（78h）→ 极弱科氏力 → 全球**绝对无台风** → 大气层流极稳定 → 翼展数百米的巨型平流层飞艇/太阳能滑翔机长期驻留 20 km 高空
- **低轨航天**：狂暴范艾伦辐射带 → 早期硅芯片频繁 SEU → 倒逼 SiC 抗辐射芯片和光学计算极早成熟
- **推进**：1.05g + 11.85 km/s → 化学火箭运力严重不足 → NTR 在航天早期（~地球 1960s）被迫商用

## Aegis 探测与磁帆技术

- Aegis 距离仅 74 万 km（地月距离 2 倍），通信延迟 ~2.5 秒 → 极早实现**实时遥控探测**
- Aegis 深层辐射带极强 → 发明**磁层帆 (Magsail)**：展开超导线圈产生人工磁场，与 Aegis 磁层高速等离子体相互作用 → 无工质减速 + 轨道悬停
- 可深入 Aegis 极区磁层和光环系统探测

## 星际探测

- **内行星**：Ember 凌星周期 ~5.4 天 → 地基望远镜凌星光谱已摸清 Crucible 大气成分 → 探测目标极明确；红外帆 + Aegis 逆向引力弹弓减速
- **外行星**：Aegis-Boreal-Glacis 1:2:4 共振链 → 相对位置极规律 → **低能量星际高速公路 (ITN)** 贯穿系统 → 离子推进 / 核聚变推进成为深空主力
- **外围天体盘**：5–10 AU 柯伊伯带类似物 + 数百 AU 奥尔特云 → 纯资源开采（水冰、He-3）+ 长基线引力波干涉测量
- **系外探索**：系统尺度极小 → 突破摄星级纳米光帆阵列加速轨道短 → 向邻近恒星系发射大量微型探测器构建传感器网络
