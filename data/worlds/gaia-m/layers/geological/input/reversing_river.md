# "赤渊"倒流河奇观

### 自变量

| 参数 | 值 | 备注 |
|------|-----|------|
| 位置 | Aegis 大陆北纬 40° 裂谷段 | 副热带超级荒漠核心区 |
| 裂谷盆地底部海拔 | −450 m | |
| 裂谷盆地面积 | 20,000 km² | |
| 地理屏障 | 高 300 m 火山岩地垒 | |
| 缺口（倒流河口） | 宽 1.5 km、长 15 km 峡谷 | ~8000 年前超级地震致地垒局部坍塌 |
| 河道长度 | 550 km | 平均坡度 ~0.08% |
| 构造沉降速率 | 20 cm/year | 裂谷活跃拉张期 |

### 因变量

| 参数 | 值 | 推导依据 |
|------|-----|---------|
| 年蒸发量 | 4.2 m/year | 超级荒漠气候 + 0.417 W/m² 地热烘烤 |
| 年入湖水量 | 84 km³/year | 4.2m × 20000 km² |
| 倒流河平均流量 | 2662 m³/s | 相当于尼罗河/科罗拉多河入海流量 |
| 涌潮特征 | 每 39 小时一次，高 30–40 m | 78 m 潮汐 + 峡谷瓶颈 → 巨型涌潮以 >40 km/h 逆流数百公里 |
| 年输入盐量 | ~30.1 亿吨 | 84 km³ × 3.5% 盐度 |
| 盐层堆积速率 | 3.5 cm/year | |
| 盐分平衡 | 沉降(20 cm/yr) >> 堆积(3.5 cm/yr) + 地下卤水暗流回流 | "地表-地下双循环" |
| 峡谷拓宽速率 | ~0.15 m/year | |
| 奇观寿命 | ~8000 年 | 峡谷拓宽至 >3 km 后决堤，裂谷被填满为深海湾 |
| 80 天年轮盐层 | 每 20.1 天交替 → 彩色条带 | 嗜极藻类不同温度下的色素沉积 |
