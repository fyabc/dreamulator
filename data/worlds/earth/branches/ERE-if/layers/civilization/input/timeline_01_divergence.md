# 分歧与中兴：避免死亡螺旋（1341–1520）

1341年安德洛尼卡三世驾崩后，东罗马帝国的两约翰内战因权臣阿波考库斯的提前死亡而缩短至不到半年。坎塔库泽努斯和平进入君士坦丁堡，保全了国库、海军与色雷斯农业区。帝国在随后近两个世纪中成功遏制塞尔维亚扩张、锁死达达尼尔海峡阻止奥斯曼渡海，并在1402年帖木儿摧毁奥斯曼后趁势收复安纳托利亚西北故地。黑死病虽造成严重人口损失，但完整的国家机器使东罗马的灾后恢复远超史实。到1520年代，帝国已转型为控制色雷斯、马其顿、希腊半岛及小亚细亚沿海的东地中海中等强权，完成了废除封建采邑、拥抱火器与重商主义的近代化转型。

## 关键事件

| 年份 | 事件 | 影响 |
|------|------|------|
| 1341 冬 | **分歧点**：阿波考库斯在君士坦丁堡暴乱中被杀 | 摄政派群龙无首，内战提前结束 |
| 1342 初 | 坎塔库泽努斯和平入城 | 国库、海军、色雷斯免遭战火 |
| 1342-1346 | 重兵驻守马其顿边境；海军封锁达达尼尔海峡 | 塞尔维亚南扩受阻；奥斯曼困于小亚细亚 |
| 1345 | 静修派（Hesychasm）确立为东正教正统 | 宗教统一增强内部凝聚力 |
| 1347 | 黑死病传入；帝国实施港口隔离制度 | 人口损失约1/3，但社会结构未崩溃 |
| 1354 | 约翰五世成年，坎塔库泽努斯和平退位 | 权力平稳交接，无二次内战 |
| 1355 | 塞尔维亚君主杜尚病死，帝国四分五裂 | 东罗马收复伊庇鲁斯、色萨利、马其顿 |
| 1402 | 帖木儿安卡拉战役彻底摧毁奥斯曼 | 奥斯曼灭亡，安纳托利亚重回碎裂状态 |
| 1402+ | 东罗马渡海收复尼西亚、普鲁萨、士麦那 | 国境推回安纳托利亚高原边缘 |
| 1501 | 萨法维波斯崛起 | 东罗马与马穆鲁克结成"联逊尼、抗什叶"同盟 |
| 1510s | 叙利亚北部战役受挫 | 黎凡特成为三方拉锯地带，东扩达到极限 |

## 内部政治

- **狂热者危机和平化解**：坎塔库泽努斯派长子马修携中央军威慑赴塞萨洛尼基，结合减免税收的温和妥协，将狂热者运动纳入市政体制
- **废除普罗尼亚制**：15世纪设立"皇家税务总局"，用货币地租取代封建包税制，解散贵族私兵，建立直属皇帝的常备军
- **宗教改革利用**：1517年宗教改革爆发后，君士坦丁堡接纳受迫害学者与人文主义者，成为"晚期文艺复兴"避风港

## 外部关系

| 势力 | 关系 | 说明 |
|------|------|------|
| 塞尔维亚 | 遏制 → 附庸化 | 杜尚死后东罗马迅速介入，将残存诸侯降为朝贡附庸 |
| 奥斯曼 | 封锁 → 灭亡 | 海军封锁海峡，帖木儿彻底摧毁 |
| 匈牙利 | 离岸平衡 | 暗中资助瓦拉几亚和保加利亚充当"肉盾" |
| 马穆鲁克埃及 | 稳固同盟 | 共同遏制萨法维波斯，获取红海通行权 |
| 法国 | 传统盟友 | 牵制神圣罗马帝国和威尼斯 |
| 莫斯科公国 | 东正教同盟 | 1472年联姻，定位为"北方东正教护教者" |
| 威尼斯/热那亚 | 竞争 | 逐步夺回黑海与爱琴海关税控制权 |

## 军事建设

- **陆军**：建立"皇家近卫火枪军"（本国东正教徒组成），大规模铸造青铜野战火炮
- **海军**：从桨帆快船向风帆战列舰（盖伦船）过渡，牢牢控制博斯普鲁斯海峡与爱琴海制海权
- **巴尔干北部挫折**：15世纪中叶匈雅提·亚诺什整合匈牙利重骑兵与胡斯派车堡火器，数次击溃东罗马干涉军 → 帝国放弃多瑙河沿岸宗主权，防线收缩至巴尔干山脉

## 文化与经济

- **重商主义转型**：大航海时代冲击地中海香料贸易后，国家垄断丝绸、玻璃、造船、火器制造；利用马穆鲁克同盟获取红海-印度洋商路
- **1351年宗教会议**：正式确立静修派为东正教正统教义
- **莫斯科"第三罗马"联动**：东罗马依然强盛，联姻变为平等的"东正教双头鹰同盟"

## 对下一时代的遗产

→ 帝国避免了"内战→引狼入室→领土丧失→沦为附庸"的死亡螺旋，以坚固的巴尔干中等强国姿态进入16世纪。

→ 废封建、拥火器、行重商主义的近代化转型为后续发展奠基，但宗教改革浪潮与外部压力即将引发新的危机。
