# 罗马共和国领土与疆域变迁

本文件记录第二罗马共和国（东罗马帝国继承国）从1341年至今的领土演变。罗马的疆域经历了中兴扩张、大分裂收缩、光复重建、帝国主义狂飙、一战剥夺、最终稳定为现代版图的漫长过程。

→ 总览见 `_overview.md`；王朝更迭见 `dynasties.md`

---

## 现代疆域（21世纪）

### 核心巴尔干

罗马共和国在欧洲的核心领土，包括色雷斯、马其顿、希腊、伊庇鲁斯及爱琴海诸岛。

| 区域 | 范围与边界 | 关键城市 |
|------|-----------|----------|
| **色雷斯** | 博斯普鲁斯海峡两侧，含君士坦丁堡 | 君士坦丁堡（首都）、阿德里安堡 |
| **马其顿** | 马其顿大部，北界斯科普里 | 塞萨洛尼基、斯科普里（边境城市） |
| **希腊/伊庇鲁斯** | 希腊本土、伊庇鲁斯 | 雅典、约阿尼纳 |
| **北方边界** | 巴尔干山脉（东段）+ 罗多彼山脉（西段） | **普罗夫迪夫**（北方边境重镇，星形棱堡群） |
| **西部边界** | 阿尔巴尼亚中南部（已罗马化，"伊利里亚大区"） | 都拉斯（亚得里亚海核心港口） |
| **爱琴海诸岛** | 基克拉泽斯群岛、纳克索斯等 | 纳克索斯 |

**北部邻国**：
- **保加利亚**：以巴尔干山脉/罗多彼山脉为界，占据多布罗加大部（含瓦尔纳、布尔加斯）。向南扩张被罗马阻挡后转向东北方向发展。
- **达契亚**（替代现实罗马尼亚）：多瑙河以北，仅拥有康斯坦察一带狭窄黑海海岸线。

→ 详见 `geo_europe.md`

### 安纳托利亚

罗马共和国在小亚细亚的核心领土，覆盖西部、中部及南部海岸。

| 区域 | 范围 | 关键城市 |
|------|------|----------|
| **西海岸** | 爱琴海沿岸 | 士麦那（Smyrna）、尼科米底亚 |
| **中部高原** | 安纳托利亚内陆 | 安卡拉、尼西亚（Nicaea） |
| **南海岸** | 托罗斯山脉以南至叙利亚边界 | 安塔利亚 |
| **东部边界** | 托罗斯山脉→幼发拉底河上游 | 埃尔祖鲁姆、穆什、比特利斯（东方堡垒群） |

→ 详见 `geo_middle_east.md`

### 高加索边界

罗马与高加索诸国的边界沿庞廷山脉和大高加索山脊延伸。

| 区域 | 归属 | 说明 |
|------|------|------|
| **庞廷山脉至特拉布宗** | 罗马 | 黑海南岸防线 |
| **巴统（Batumi）** | **罗马** | 格鲁吉亚人聚居区，但归属罗马；黑海东岸不冻深水港，黑海舰队可威慑高加索 |
| **大高加索山脊** | 罗马-格鲁吉亚/亚美尼亚界山 | 自然地理分界线 |

**亚美尼亚边界**：
- 卡尔斯（归亚美尼亚）— 阿勒山（界山，两国共有）— 凡城（归亚美尼亚）
- 凡湖由两国分管；罗马保留埃尔祖鲁姆、穆什等安纳托利亚东部堡垒群
- 亚美尼亚为高加索"平衡手"，在罗马资本、俄罗斯能源与波斯贸易间走钢丝

**格鲁吉亚关系**：失去巴统出海口，经济与军事上高度依附罗马，坚定亲罗/亲西方阵营。

### 叙利亚

罗马共和国在中东的核心领土，构成高度工业化的"叙利亚大区"。

| 要素 | 详情 |
|------|------|
| **控制线** | 迪亚巴克尔 → 拉卡 → 阿勒颇 → 霍姆斯 → 拉塔基亚 |
| **地理逻辑** | 经典"沙漠-绿洲分界线"，控制安纳托利亚东南部水源+叙利亚北部绿洲 |
| **归属罗马** | 迪亚巴克尔、拉卡、阿勒颇、霍姆斯、拉塔基亚 |
| **归属叙利亚** | **塔尔图斯**、大马士革、贝鲁特、的黎波里（黎巴嫩地区整体归入叙利亚） |

**叙利亚共和国现状**：因黎巴嫩未独立，叙利亚成为微缩中东——马龙派控制贝鲁特，逊尼派占据大马士革，阿拉维派盘踞拉塔基亚/塔尔图斯。自1950年代以来爆发三次内战，罗马以"保护东正教/基督徒侨民"为由常年军事介入北部。

→ 详见 `geo_middle_east.md`

## 历史领土变迁

| 年代 | 事件 | 领土变化 |
|------|------|----------|
| 1341-1354 | 两约翰内战结束 | 坎塔库泽努斯统一色雷斯、马其顿 |
| 1402后 | 帖木儿摧毁奥斯曼 | 光复小亚细亚腹地，国境推回安纳托利亚 |
| 1450s | 肃清爱琴海拉丁势力 | 粉碎纳克索斯公国，纳入基克拉泽斯群岛 |
| 1485 | 《干地亚条约》 | 从威尼斯赎回克里特 |
| 1489 | 紫室联姻 | 和平接管塞浦路斯 |
| 1510 | "反向十字军" | 驱逐罗德岛骑士团至马耳他，控制十二群岛 |
| 1522 | 海外扩张 | 建立巴西殖民地（紫木之地/Porphyria） |
| 1571 | 紫室之劫 | 君士坦丁堡沦陷，帝国分裂，部分岛屿短暂丢失 |
| 1580-1645 | 光复战争 | 流亡政权以塞萨洛尼基、克里特为基地，1645年打回君堡 |
| 17-18世纪 | 黑海拉锯 | 与沙俄争夺克里米亚/黑海北岸，勉强守住刻赤、赫尔松 |
| 1812 | 《黑海密约》 | **主动放弃克里米亚及黑海北岸全部据点**，换取博斯普鲁斯海峡"内水"地位 |
| 1824 | 殖民地独立 | 巴西/紫木之地独立 |
| 1880s | 帝国主义扩张 | 夺取北非（昔兰尼加、的黎波里塔尼亚）、东非（厄立特里亚沿海） |
| 1895-1900 | 远东扩张 | 强租"新查士丁港"（华南/南洋微型加煤站） |
| 1914 | 一战爆发 | 远东租界被英日海军第一周即占领 |
| 1916.12.11 | 紫室倒戈 | 背刺同盟国，转投协约国 |
| 1918 | 一战末期 | 击败叙利亚叛军推进至大马士革郊外，但被迫撤回 |
| 1919 | 巴黎和会 | **全部海外殖民地被英法剥夺**；失去耶路撒冷与南叙利亚 |
| 1922 | 紫室风暴 | 第二罗马共和国成立，领土稳定为现代版图 |

## 已失领土

### 克里米亚（1812年主动割让）

- 14-16世纪：通过沿海堡垒（赫尔松、刻赤）和"以夷制夷"策略维持点状控制
- 17-18世纪：与沙俄在第聂伯河下游、亚速海沿岸爆发惨烈"黑海边境战争"
- **1812年《黑海密约》**：罗马全面撤出克里米亚半岛及黑海北岸所有据点，默认沙俄吞并；换取博斯普鲁斯海峡"内水"地位及多瑙河下游、巴尔干山脉以南的绝对控制权
- 历史定性："断臂求生"——甩掉北方陆上包袱，集中力量经营东地中海与中东
- 现代立场：罗马明确不寻求重返黑海北岸，利用海峡控制权实施"离岸平衡"

### 巴西/紫木之地（1824年独立）

- 1522年建立，是罗马在海外最大的殖民尝试
- 梅利塞诺斯王朝时期痛苦失去，标志帝国第一次大规模殖民地丧失

### 北非（1919年巴黎和会剥夺）

- 昔兰尼加与的黎波里塔尼亚（今利比亚）：1880年代占领，打造为中地中海海军堡垒
- 巴黎和会上英法以"民族自决"为借口煽动阿拉伯部落起义，操纵国际联盟将其"委任"给意大利与法国
- 成为二战中罗马与意大利在北非沙漠死斗的历史伏笔

### 东非（1919年巴黎和会剥夺）

- 厄立特里亚沿海（马萨瓦）及索马里兰边缘：1890年代以"寻找祭司王约翰领地"为借口扩张
- 英国以"保障苏伊士运河红海侧翼绝对安全"为由强行接管，并入英属东非或意属厄立特里亚
- 罗马彻底失去红海沿岸立足点，仅退守亚历山大港飞地

### 远东（1919年巴黎和会剥夺）

- "新查士丁港"：1895/1900年强租的微型加煤站与通商口岸
- 一战爆发第一周即被英日海军占领
- 巴黎和会上英法日以"战时敌产已合法没收"为由转交中国政府或英国海峡殖民地
- 标志罗马彻底放弃"全球海洋帝国"幻梦，战略重心全面收缩回东地中海

### 耶路撒冷与南叙利亚（一战丧失）

- 19世纪后半叶帝国主义时代夺取
- 一战末期击败叙利亚叛军推进至大马士革郊外，但被迫撤回
- 巴黎和会上英法拒绝交出耶路撒冷，圣城成为国际联盟/联合国**独立实体**（Corpus Separatus），由联合国、罗马（东正教代表）、梵蒂冈（天主教代表）与伊斯兰瓦克夫四方共管
- 罗马在圣墓教堂周边保留事实"治外法权区"，驻扎"圣墓卫队"

## 东地中海岛屿（一直控制）

| 岛群 | 获取方式 | 1571年紫室之劫 | 现代地位 |
|------|----------|----------------|----------|
| **克里特** | 1485年《干地亚条约》从威尼斯赎回 | 短暂被西班牙-威尼斯联军夺回，1640年代光复战争中为流亡政权海军基地 | 南部海军枢纽，苏达湾部署AIP潜艇基地 |
| **塞浦路斯** | 1489年紫室联姻和平接管 | 未受波及（距君堡较远） | 最富裕海外大区之一，EMGF天然气指挥中心 |
| **基克拉泽斯/纳克索斯** | 1450年代武力粉碎纳克索斯公国 | 1571年被西班牙舰队洗劫，1580年代收复 | 高端旅游业与古典考古学核心地带 |
| **十二群岛（罗德岛等）** | 1510年"反向十字军"驱逐骑士团 | 未受波及 | 监视安纳托利亚南岸与黎凡特北部的雷达前哨与空军基地 |
