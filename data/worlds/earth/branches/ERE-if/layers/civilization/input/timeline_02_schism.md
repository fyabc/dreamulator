# 大分裂与海外罗马（1520–1650）

16世纪中叶，宗教改革浪潮催生了国内"新破坏圣像派"运动，皇帝借势没收修道院地产以充军费，引发大牧首绝罚与"修道院战争"内战。1571年君士坦丁堡特大地震与西班牙-威尼斯联军登陆双重打击下，帝国陷入堪比1204年的"大分裂"——四个政权割据，首都沦为西班牙傀儡。然而流亡政权在塞萨洛尼基和摩里亚被迫推行深度近代化改革：召开首次"全国等级会议"、引入莫里斯式线式战术、建设科学研究中心。与此同时，1522年建立的巴西殖民地"紫木之地"在本土危机中吸收大量难民与学者，崛起为"海外罗马"。1645年借三十年战争之机光复君士坦丁堡，1652年《紫室宪章》确立君主立宪制，近毁灭的经历反而成为帝国现代化的催化剂。

## 关键事件

| 年份 | 事件 | 影响 |
|------|------|------|
| 1520s-1550s | 巴西殖民地建立（"Terra Porphyria / 紫木之地"） | 为日后"海外罗马"奠基 |
| 1558 | 安德洛尼卡四世颁布《世俗化敕令》 | 没收修道院地产，大牧首绝罚皇帝 |
| 1558-1571 | **修道院战争**（"新破坏圣像派"vs保守教会） | 堪比法国胡格诺战争的惨烈内战 |
| 1571 秋 | 君士坦丁堡特大地震 + 西班牙-威尼斯联军登陆 | **"紫室之劫"**——帝国再次失去首都 |
| 1571-1575 | **帝国大分裂为四个政权** | 见下表 |
| 1585 | 流亡政权召开首次"全国等级会议" | "无代表不纳税"原则确立 |
| 1590s | "新罗马军团"采用莫里斯式线式战术 | 以少胜多击溃匈牙利与西班牙旧式军队 |
| 1618 | 三十年战争爆发 | 西班牙无力顾及东地中海 |
| 1630s | 光复战争发动（法国黎塞留秘密资助） | 线式战术军队反攻 |
| 1641 | 色雷斯战役击溃西班牙-威尼斯联军 | 光复战争决定性胜利 |
| 1645 | **皇室还都君士坦丁堡** | 大分裂时代结束 |
| 1648 | 《威斯特伐利亚和约》承认东罗马主权 | 帝国重回欧洲国际体系 |
| 1652 | **《紫室宪章》颁布** | 确立二元制君主立宪 |

## 大分裂四政权（1571-1645）

| 政权 | 中心 | 性质 | 说明 |
|------|------|------|------|
| 君士坦丁堡政权 | 君士坦丁堡 | 西班牙-威尼斯傀儡 | 寡头共和国，天主教东方前哨 |
| 塞萨洛尼基政权 | 塞萨洛尼基 | 正统皇室流亡 | 控制马其顿部分地区，改革中心 |
| 摩里亚专制国 | 伯罗奔尼撒 | 军事独裁 | 新破坏圣像派贵族 + 地方军阀 |
| 特拉比松帝国 | 黑海沿岸 | 独立 | 孤立无援 |

## 内部改革

### 政治近代化
- **全国等级会议（1585）**：流亡皇帝为重商家族与手工业行会妥协，确立"无代表不纳税"，标志从绝对君主制向早期君主立宪过渡
- **封建采邑制彻底废除**：旧贵族和保守教会势力在分裂中被摧毁

### 军事革命
- 高薪聘请荷兰与德国雇佣兵教官（经历八十年战争的 veterans）
- 全面引入**莫里斯式长矛-火枪线式方阵**
- 建立标准化野战火炮兵工厂
- "新罗马军团"在1590年代巴尔干防御战中多次以少胜多

### 科学启蒙
- 摩里亚和塞萨洛尼基大学成为欧洲受迫害学者避难所
- 哥白尼日心说、伽利略力学理论公开讲授
- 东罗马迎来"晚期文艺复兴"与早期科学革命交汇

## 海外罗马的崛起

| 阶段 | 时间 | 说明 |
|------|------|------|
| 殖民开拓 | 1520s-1550s | 绕过好望角，在巴西东海岸建商站与殖民地 |
| 难民涌入 | 1571+ | 亲王携残存舰队、宫廷学者、工匠、国库黄金跨大西洋逃亡 |
| 克里奥尔文化 | 1571-1620 | 吸收新教徒、犹太商人、非洲奴隶，形成高度商业化、宽容的"克里奥尔罗马文化" |
| 本土输血 | 1580s-1640s | 蔗糖、烟草、巴西木贸易输送海量财富；火枪、战舰、法律草案大量来自巴西 |

- **"新君士坦丁堡"**（Nea Konstantinoupolis）：位于今萨尔瓦多附近，成为复国运动最大资金来源与近代思想孵化器

## 外部关系

| 势力 | 关系变化 | 说明 |
|------|----------|------|
| 西班牙哈布斯堡 | 占领者 → 被驱逐 | 深陷尼德兰八十年战争，被东罗马暗中消耗 |
| 法国 | 铁杆盟友 | 波旁王朝亨利四世为对抗哈布斯堡，持续输送火炮与资金 |
| 俄罗斯 | 坚定后盾 | 混乱时代后罗曼诺夫王朝视东罗马为"受难宗主与正统灯塔" |
| 波兰-立陶宛 | 无暇南顾 | 深陷东欧平原泥潭，客观为摩里亚喘息提供空间 |
| 匈牙利 | 降级为哈布斯堡附庸 | 宗教改革内部分裂，被神罗蚕食 |
| 萨法维波斯 | 东方威胁 | 夺取尼西亚和士麦那，但无力跨越海峡消灭流亡政权 |
| 马穆鲁克埃及 | 同盟崩溃 | 16世纪中叶因内乱与航路衰落而崩溃，东罗马黎凡特缓冲丧失 |

## 文化

- **东罗马流亡文学**：分裂时期产生的流亡诗歌与政治哲学深刻影响后世国家认同
- **"罗马私掠许可证"**：流亡政权暗中资助尼德兰新教徒与英国私掠船消耗西班牙海军
- **东正教理性化萌芽**：远离传统教条束缚的大学成为科学革命传播中心

## 对下一时代的遗产

→ 帝国以"主权国家"身份重返欧洲牌桌，确立了君主立宪与重商主义并行的二元体制。

→ 巴西殖民地经济体量逼近本土，跨大西洋帝国联邦的隐患即将浮现。

→ 俄罗斯从生死盟友转为地缘竞争者的裂变正在酝酿。
