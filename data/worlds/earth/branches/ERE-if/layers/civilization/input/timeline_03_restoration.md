# 复兴与启蒙：宪政帝国的黄金世纪（1650–1800）

1652年《紫室宪章》确立了"皇权统帅军队与外交、议会掌控财政与立法"的二元制君主立宪，扫除了绝对君主制的财政崩溃隐患。东正教通过"新帕拉马斯主义"改革调和理性与神秘主义，君士坦丁堡皇家科学院（1660）成为与英国皇家学会齐名的学术中心。外交上上演堪比1756年"外交革命"的阵营大反转——与奥地利哈布斯堡结盟遏制俄罗斯南下。军事上经历纳迪尔沙三次冲击后以星形棱堡与海军封锁取胜，东方边界巩固至托罗斯山脉。18世纪末开明专制改革废除农奴制、颁布《新查士丁尼法典》，法国大革命爆发后帝国成为抵御激进共和主义的"东方防波堤"。

## 宪政体制

### 《紫室宪章》（1652）
- **皇权范围**：军队统帅、外交决策
- **议会范围**：财政拨款、立法审议
- **性质**：类似17世纪末英国或后来的德意志帝国，二元制君主立宪

### 巴西危机与妥协
- 17世纪末巴西殖民地经济体量逼近本土，精英以"无代表不纳税"抗税
- **1692年《萨尔瓦多法案》**：巴西升格为共主邦联内平等王国，君士坦丁堡帝国议会保留20个专属席位
- 避免了类似美国独立战争的流血冲突，将巴西资源绑定于帝国

## 外交革命

### 俄罗斯：从盟友到死敌

| 阶段 | 关系 | 说明 |
|------|------|------|
| 流亡时期 | 生死盟友 | 罗曼诺夫王朝为东罗马最坚定后盾 |
| 1654+ | 裂变 | 俄罗斯吞并左岸乌克兰后意识到：强大的东罗马是其巴尔干扩张与"第三罗马"宣称的最大法理障碍 |
| 1680s | 死敌 | 俄罗斯暗中支持巴尔干斯拉夫民族主义，试图建立"斯拉夫东正教联盟" |

### 东罗马-奥地利同盟

- 1680年代签订《维也纳-君士坦丁堡密约》
- 东罗马承认奥地利在匈牙利和巴尔干北部的势力范围
- 换取奥地利在对抗俄罗斯时的军事支持
- **意义**：彻底宣告中世纪"宗教十字军"时代终结，确立"国家利益至上"的近代外交准则

### 东欧缓冲区

- **多瑙河-巴尔干东正教共主联邦**：对塞尔维亚、瓦拉几亚、摩尔达维亚、阿尔巴尼亚实行"开明帝国主义"
- 派遣法纳尔文官治理多瑙河公国，给予当地贵族东正教自治权与关税优惠
- 阿尔巴尼亚等山地地区因宗教与地形原因长期成为走私与游击战温床
- 奥地利哈布斯堡对巴尔干西北部保有领土野心，东罗马采取时战时和策略

## 军事

### 纳迪尔沙冲击（1730-1747）

| 战役 | 结果 |
|------|------|
| 1735 阿尔汉格尔战役 | 东罗马惨败，线式方阵在崎岖地形被波斯轻骑兵侧翼包抄 |
| 1680s 亚美尼亚高原战役 | 补给线过长，遭游击战与轻骑兵袭扰 |
| 棱堡防御战 | 改守星形棱堡体系 + 海军封锁切断波斯后勤 → 成功 |
| 1747+ 反攻 | 纳迪尔沙遇刺后收复失地，边界永久巩固至托罗斯山脉与幼发拉底河上游 |

**教训**：笨重线式方阵不适合崎岖地形野战，棱堡+制海权的防御战略更有效。

### 爱奥尼亚海战（1688）
- 威尼斯趁东罗马主力东调，联合法国舰队重创海军
- 丧失伯罗奔尼撒南部部分要塞与爱琴海部分岛屿
- 暴露"重陆轻海"隐患 → 帝国议会通过《海军扩建法案》

## 文化与思想

### 希腊启蒙运动
- **法纳尔知识分子**（Phanariotes）为思想先锋
- 将古典哲学"理性"（Logos）与东正教"神化"（Theosis）融合
- 核心理念："探索自然法则即是领悟上帝之光"

### 新帕拉马斯主义
- 神学家将理性主义与东正教神秘主义结合
- "探索自然规律即是赞美上帝"
- 类似"新教伦理"的宗教改革，扫除资本主义与近代科学的思想障碍
- 阿索斯山修道院改造为神学与自然科学并重的研究院

### 开明专制（1780s）
- 废除色雷斯和安纳托利亚残存农奴制
- 颁布《新查士丁尼法典》统一全国法律
- 对天主教徒和犹太人实行宗教宽容
- 触动传统军事贵族与保守教会利益，国内"改革派"与"保守派"剧烈摇摆

## 经济

- **18世纪末**：巴西殖民地经济体量占帝国总量近四成
- 帝国本土人口约800万（1700年），加上巴西数百万
- 君士坦丁堡成为与巴黎、伦敦齐名的启蒙运动前夕思想与科学重镇

## 对下一时代的遗产

→ 帝国蜕变为具有全球影响力的近代化列强，欧洲"均势外交"中的"第四极"。

→ "新罗马派"激进地下组织在中产阶级与青年军官中悄然成型，卢梭《社会契约论》已秘密传入。

→ 巴西精英要求完全平等代表权，跨大西洋帝国纽带面临撕裂风险。
