# 革命时代：从王朝帝国到公民国家（1800–1850）

1799年激进军官与法纳尔知识分子发动政变，成立短暂的"新罗马共和国"，恢复古罗马政治称谓（保民官、公民大会），但在英俄封锁下三年即亡。1802年宪政妥协保留议会与公民权利，共和理念植入帝国政治基因。随后巴尔干民族主义狂飙撕裂帝国北方领土，"希腊民族国家"与"公民罗马主义"的认同之争贯穿上半世纪。巴西殖民地援引罗马行省平等传统赢得独立，倒逼本土工业化转型。1848年"欧洲之春"引发三方内战，但沙俄趁火打劫时"公民军团"在阿德里安堡击退俄军——与曼齐刻尔特后贵族争相投敌形成鲜明对比。1850年《新查士丁尼宪法》确立君主立宪、两院制议会、公民平等与政教分离，帝国完成了从王朝帝国到近代公民国家的艰难蜕变。

## 关键事件

| 年份 | 事件 | 影响 |
|------|------|------|
| 1799-1802 | **"新罗马共和国"** | 激进军官+法纳尔知识分子政变，恢复保民官/公民大会 |
| 1802 | **《1802年宪政妥协案》** | 共和国覆亡，但保留议会与部分公民权利 |
| 1804-1835 | **巴尔干民族起义** | 塞尔维亚、保加利亚、瓦拉几亚、阿尔巴尼亚相继武装起义 |
| 1815-1824 | **巴西独立战争** | "新罗马尼亚共和国"独立，《新君士坦丁堡和约》承认 |
| 1820-1840s | **工业革命传入** | 纺织/造船/兵工兴起于塞萨洛尼基、尼科米底亚、士麦那 |
| 1828-1829 | 俄拜边境冲突 | 高加索主山脊划定边界，沙俄南下转向中亚与远东 |
| 1839-1841 | **夺取亚历山大港** | 趁埃及继承危机控制港口，改造为东地中海核心加煤站 |
| 1845 | 君士坦丁堡-阿德里安堡铁路通车 | 帝国第一条铁路 |
| 1848 | **"欧洲之春"** | 三方内战：保皇派 vs 共和派 vs 联邦派 |
| 1849 | **阿德里安堡战役** | "公民军团"依托铁路与线膛枪重创俄军先锋 |
| 1850 | **《新查士丁尼宪法》** | 君主立宪、两院制、公民平等、政教分离 |

## "新罗马共和国"（1799-1802）

- 受法国大革命与拿破仑东征直接影响
- 废除贵族特权，恢复古罗马政治称谓
- 设立"公民大会"、选举"保民官"（Tribunus）以否决专制法令
- 试图与法兰西第一共和国缔结"共和国同盟"
- **崩溃原因**：英俄联合舰队封锁海峡，保守教会与安纳托利亚地主强烈反弹，经济崩溃
- **遗产**："公民权""保民官""共和国"等古典-启蒙混合词汇植入帝国政治基因

## 民族主义与领土收缩

### 巴尔干联邦解体

| 地区 | 起义时间 | 结果 |
|------|----------|------|
| 塞尔维亚 | 1804- | 武装起义 → 自治 |
| 保加利亚 | 1810s-1820s | 武装起义 |
| 瓦拉几亚 | 1820s | 独立 |
| 阿尔巴尼亚 | 1820s-1835 | 自治/独立 |

- **残酷民族仇杀**：马其顿与色雷斯边境地带希腊裔、斯拉夫裔、阿尔巴尼亚裔相互清洗，数十万平民流离失所
- **至1835年**：帝国实际控制线南撤至马其顿南部与色雷斯平原

### 认同危机
- **保守派**：主张"希腊民族国家"路线，强调语言与血统
- **革命派与城市资产阶级**：高举**"公民罗马主义"（Civic Romanism）**，援引查士丁尼法典，主张"凡效忠宪法、纳税服役者皆为罗马公民"
- 1848年革命后公民罗马主义凭借跨族群动员力占据上风

### 高加索与安纳托利亚东部
- 亚美尼亚人要求文化自治与教会独立
- 库尔德部落抗拒中央征税与征兵
- 土库曼与切尔克斯人在俄拜夹缝中求生
- 东罗马通过"军事屯垦+地方自治"混合政策维系多民族均势
- **1828-1829年俄拜冲突**：俄军无法突破埃尔祖鲁姆-特拉布宗火力网，双方沿高加索主山脊划定边界

## 殖民地独立

### 新罗马尼亚独立战争（1815-1824）

| 要素 | 说明 |
|------|------|
| 殖民地诉求 | 援引古罗马"行省平等权"传统与"无代表不纳税"原则，要求帝国议会完全平等席位 |
| 战争经过 | 持续九年，东罗马远洋舰队屡次封锁海岸但补给线过长 |
| 结局 | 1824年《新君士坦丁堡和约》承认"新罗马尼亚共和国"独立 |
| 保留条款 | 最惠国贸易待遇与海军补给权 |
| 影响 | 切断海外输血，倒逼本土经济结构转型 |

## 工业革命

- **1820-1840s**：引进英国蒸汽机与比利时冶金技术
- 塞萨洛尼基、尼科米底亚、士麦那兴起纺织、造船、兵工产业
- **1845年**：君士坦丁堡至阿德里安堡铁路通车（帝国第一条）
- 工业化催生城市资产阶级与产业工人阶级，为1848革命提供社会基础

## 1848 "欧洲之春"

### 三方内战
- **保皇派**：传统军事贵族与保守教会
- **共和派**：学生、工人、自由派军官、开明贵族
- **联邦派**：地方自治诉求者
- 诉求：废除书报审查、普选、分配教会与贵族闲置土地

### 沙俄干涉与"公民军团"

| 对比 | 曼齐刻尔特（1071） | 阿德里安堡（1849） |
|------|---------------------|---------------------|
| 外敌入侵时 | 地方贵族争相投靠突厥人 | 公民民族主义空前动员 |
| 军队组成 | 封建征召兵 | 城市民兵+农民自卫队+革命军官 → "罗马公民军团" |
| 技术基础 | 无 | 铁路补给+线膛枪火力 |
| 结果 | 帝国崩溃 | 重创俄军先锋，沙俄被迫撤军 |

- **"罗马公民军团"（Legiones Civium）**：保民官与临时执政官指挥
- 欧洲列强震惊于"古老帝国"的近代国民战争潜力

## 文化与思想

### 新古典浪漫主义
- 受温克尔曼、歌德、雪莱等西欧浪漫主义学者影响
- 部分激进文人尝试在沙龙与大学中重建"多神教仪式"与"祖先崇拜"
- 教会初以"异端复辟"谴责

### 新帕拉马斯综合主义
- 法纳尔神学家将古典哲学重新诠释为**"福音的预备"（Praeparatio Evangelica）**
- 强调希腊理性与东正教灵修的连续性
- 国家介入调停：部分古典节日改造为"公民纪念日"，公共建筑融合东罗马穹顶与新古典柱式
- 催生19世纪中叶东罗马文学、建筑与音乐的"浪漫-古典黄金期"

### 亚历山大港的夺取（1839-1841）
- 埃及穆罕默德·阿里王朝陷入继承危机与军阀混战
- 东罗马蒸汽动力舰队介入尼罗河三角洲
- 电报通讯、蒸汽航运、近代市政管理使帝国不再重蹈中世纪"跨海统治难"覆辙
- 改造为东地中海核心加煤站、自由贸易港与海军基地
- 有效弥补巴西独立后的财政缺口

## 1850年帝国全景

| 维度 | 状态 |
|------|------|
| 领土 | 色雷斯、马其顿南部、希腊半岛、爱琴海诸岛、小亚细亚中西部、亚历山大港 |
| 人口 | 约1100万 |
| 体制 | 君主立宪，两院制议会，公民兵役制，政教协同但世俗法律至上 |
| 经济 | 早期工业化完成，铁路网初具规模，亚历山大港与君堡双枢纽贸易体系 |
| 陆军 | 全面换装线膛枪与野战炮，公民征兵制 |
| 海军 | 蒸汽护卫舰与早期铁甲舰原型，东地中海制海权 |
| 意识形态 | 公民罗马主义为核心国家认同，东正教完成理性化转型 |

## 对后续时代的遗产

→ 帝国在民族主义撕裂、殖民体系崩塌、工业革命冲击与1848革命烈火中完成向近代公民国家的制度转型。

→ 失去巴尔干北部与跨洋殖民地，换来近代公民国家骨架、工业化引擎与国民凝聚力。

→ "公民罗马主义"提供了跨族群的国家认同基础，但巴尔干民族国家的独立为未来地缘博弈埋下伏笔。
