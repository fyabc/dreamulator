# 五大王朝更迭全景

东罗马（东罗马帝国）在本时间线中避免了1453年灭亡，皇位继承奉行 **"强者居之"与"法统重塑"** 的罗马传统，而非西欧的"血统绝对神圣"。从1341年到1922年，帝国经历了五个截然不同的大王朝，每一次更迭都是帝国面对**宗教、启蒙、民族、阶级**四大现代性危机时的体制重塑。

→ 总览见 `_overview.md`；领土变迁见 `territory.md`

## 一、帕莱奥洛根王朝 (Palaiologos, 1261-1354)

**"中兴的余晖与正统的断绝"**

### 更迭逻辑

1341年"两约翰内战"中，约翰六世·坎塔库泽努斯没有选择妥协退位，而是在1354年以"勾结拉丁人、危害东正教正统"为由，彻底废黜并刺瞎了年幼的约翰五世·帕莱奥洛格斯。帕莱奥洛根家族主支被流放至摩里亚，彻底失去皇位继承权。

### 历史遗产

- 留下了收复色雷斯和马其顿的底子
- 《查士丁尼法典》的近代化修订草案
- 晚期软弱统治成为后世"反面教材"

---

## 二、坎塔库泽努斯王朝 (Kantakouzenos, 1354-1575)

**"安纳托利亚的雄狮与宗教改革的殉道者"**

### 崛起背景

作为帝国最大的地主与军事贵族，坎塔库泽努斯家族代表安纳托利亚本土派（希腊-突厥混血军事贵族）利益。带领帝国挺过14世纪黑死病，并利用帖木儿击溃奥斯曼的契机（1402年安卡拉战役），将国境线重新推回小亚细亚腹地。

### 核心特征

极度崇尚古典军国主义与东正教神秘主义的王朝。

### 关键成就与事件

- 建立强大的"新罗马军团"，15世纪主导"希腊启蒙"早期萌芽
- 1402年后光复小亚细亚腹地，国势中兴
- 1522年建立巴西殖民地（紫木之地/Porphyria），开启海外扩张
- 15世纪肃清爱琴海拉丁人势力（1450年代粉碎纳克索斯公国，1485年赎回克里特，1489年和平接管塞浦路斯，1510年驱逐罗德岛骑士团）
- 1571年君士坦丁堡特大地震+西班牙-威尼斯联军趁虚而入（紫室之劫），末代皇帝在圣索菲亚大教堂穹顶下自焚殉国

### 覆灭

16世纪中叶，面对西方宗教改革冲击，末代皇帝试图强行没收修道院财产以扩充海军，引发惨烈的**修道院战争**。1571年君士坦丁堡大地震后，西班牙-威尼斯联军发动紫室之劫。主支断绝，帝国陷入大分裂。

## 三、拉斯卡里斯-科穆宁王朝 (Laskaris-Komnenos, 1580-1799)

**"流亡的复国者与启蒙时代的开明专制"**

### 崛起背景

君士坦丁堡沦陷后，一位拥有尼西亚帝国（拉斯卡里斯）和特拉比松（科穆宁）古老血统的地方军阀，在塞萨洛尼基整合流亡政权。通过联姻与法理宣称，将两个代表"东罗马抵抗精神"的古老姓氏结合，建立新王朝。

### 核心特征

充满悲情色彩与外交智慧的"流亡-复国"王朝。

### 关键成就与事件

- 向法国和波斯借力，1640年代成功光复君士坦丁堡（1645年）
- 克里特在光复战争中成为流亡政权最重要的海军基地与粮仓
- 18世纪推行"开明专制"，主导东正教与科学革命的"理性调和"
- 17-18世纪与沙俄争夺黑海北岸，勉强守住刻赤和赫尔松

### 覆灭

受法国大革命思潮冲击，1799年君士坦丁堡爆发激进军官与法纳尔知识分子的联合政变。末代皇帝因拒绝放弃绝对君权，被推翻（处决或流放俄国）。**"新罗马共和国"** 短暂建立，君主制遭遇自1204年以来第一次法理中断。

---

## 四、梅利塞诺斯王朝 (Melissenos, 1802-1850)

**"安纳托利亚的军阀与民族主义的绞肉机"**

### 崛起背景

1802年，短暂的共和国因经济崩溃和英俄联合舰队封锁而倒台。掌握军权的安纳托利亚大军区将领（梅利塞诺斯家族，小亚细亚传统军事贵族）发动"紫室复辟"，黄袍加身。

### 核心特征

典型的近代军人独裁王朝。

### 关键成就与事件

- 凭借刺刀维持摇摇欲坠的多民族帝国
- 1812年《黑海密约》：主动放弃克里米亚与黑海北岸，换取博斯普鲁斯海峡"内水"地位（"断臂求生"）
- 残酷镇压巴尔干民族起义（"新保加利亚屠夫"）
- 1820年代痛苦失去巴西殖民地（1824年独立）

### 覆灭

面对1848年"欧洲之春"革命狂潮与沙俄趁火打劫，梅利塞诺斯旧式军队一触即溃。自发组织的"公民军团"在阿德里安堡击退俄军。战后，愤怒的资产阶级和军方少壮派彻底抛弃腐朽皇室，逼迫末代皇帝签署《新查士丁尼宪法》后退位。

---

## 五、卡波季斯第亚斯王朝 (Kapodistrias, 1850-1922)

**"立宪的紫室与残缺的胜利"**

### 崛起背景

1850年，击退沙俄干涉后，帝国议会推举在抗俄战争中立下赫赫战功、且深受法纳尔文官集团支持的近代化将领/政治家为"罗马人的皇帝与独裁官"。标志帝国正式确立**二元制君主立宪制**。

### 核心特征

带有浓厚资产阶级色彩、拥抱工业革命与帝国主义的"近代王朝"。

### 关键成就与事件

- 修建东地中海大铁路，推动工业化
- 1880年代主导瓜分非洲狂潮（夺取亚历山大港、北非利比亚、东非厄立特里亚）
- 远东设立微型租界"新查士丁港"
- 外交长袖善舞，使罗马跻身"次级列强"
- 1916年12月11日"紫室倒戈"：背刺同盟国转投协约国，在巴尔干前线发动突袭

### 覆灭

一战中帝国虽倒戈协约国并浴血奋战，却在1919年巴黎和会上被英法背刺，彻底失去耶路撒冷与南叙利亚，全部海外殖民地被剥夺。"残缺的胜利"引爆国内复仇主义狂怒与战后经济崩溃。1922年秋，激进共和派与左翼联盟发动"紫室风暴"，末代皇帝在英军战舰护送下流亡伦敦。**延续近两千年的君主制宣告彻底终结，"第二罗马共和国"在废墟中诞生。**

## 王朝更迭的底层逻辑

每一次王朝的覆灭，都不是简单的改朝换代，而是帝国在面临四大现代性危机时通过流血完成的体制重塑：

1. **帕莱奥洛根**（旧贵族）— 亡于内战与软弱
2. **坎塔库泽努斯**（安纳托利亚大地主）— 亡于宗教改革与西方干涉
3. **拉斯卡里斯-科穆宁**（流亡军阀/古老法统）— 亡于启蒙思想与法国大革命冲击
4. **梅利塞诺斯**（小亚细亚传统军方）— 亡于19世纪民族主义与公民觉醒
5. **卡波季斯第亚斯**（法纳尔官僚/立宪派）— 亡于20世纪的帝国主义背叛与大众政治的狂飙

> 当1922年最后一位皇帝离开君士坦丁堡时，罗马人终于意识到：**能保卫罗马的不是紫室里的血统，而是宪法、公民与履带。**

## 政治制度演变

| 时期 | 政体形态 | 特征 |
|------|----------|------|
| 1341-1575 | 绝对君主制 | 皇帝集权，元老院咨询，军事贵族主导 |
| 1580-1799 | 开明专制 | 皇帝仍握实权，法纳尔文官崛起，启蒙改革 |
| 1799-1802 | 第一共和国 | 短暂存在，经济崩溃+英俄封锁，迅速倒台 |
| 1802-1850 | 军人独裁 | 刺刀维持多民族帝国，旧式军队为核心 |
| 1850-1922 | 二元制君主立宪 | 《新查士丁尼宪法》，议会与皇帝分权，资产阶级参政 |
| 1922-至今 | 第二罗马共和国 | 议会制共和+强总统，宪法至上，公民普选 |

## 关键机构

### 元老院/议会 (Senate → Parliament)

- **东罗马时期**：元老院（Σύγκλητος/Synkletos）为咨询机构，成员为贵族与高级教士
- **1850年后**：《新查士丁尼宪法》将其改组为两院制议会——上院（元老院）由世袭贵族与皇帝任命组成，下院（公民院）由财产资格选举产生
- **1922年后**：共和国实行一院制国民议会（Βουλή/Boule），全民普选

### 保民官 (Tribune)

- 古典传统的延续与复兴
- 1848年革命后设立"公民保民官"职位，代表平民利益监督政府
- 共和国时期保留为宪法法院的前身机构

### 军事组织

| 时期 | 军队形态 |
|------|----------|
| 1341-1575 | 新罗马军团：封建军事贵族+安纳托利亚本土派 |
| 1580-1799 | 常备军+外国雇佣兵（法国/波斯顾问训练） |
| 1802-1850 | 旧式军队为核心，镇压民族起义 |
| 1848后 | 公民军团崛起，与旧军并存 |
| 1850-1922 | 近代化国防军，普鲁士顾问协助改革 |
| 1922-至今 | 北约标准现代化军队，自主国防工业 |
