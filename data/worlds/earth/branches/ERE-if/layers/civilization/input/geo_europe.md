# 欧洲地缘

在这条没有奥斯曼帝国征服、东罗马成功延续并主导巴尔干半岛的时间线中，现代巴尔干版图呈现出"十字军式"地缘撕裂。由于**伊斯兰势力彻底缺席**，巴尔干没有波黑穆斯林，也没有阿尔巴尼亚的大规模伊斯兰化。国界线完全是**东正教（罗马/保加利亚/塞尔维亚）、天主教（克罗地亚/哈布斯堡）与本土异端（波斯尼亚）**经过七百年血腥拉锯后形成的"地质断层"。

→ 总览见 `_overview.md`；中东与高加索地缘见 `geo_middle_east.md`；海外领地见 `geo_overseas.md`

## 一、巴尔干诸国

### 保加利亚共和国（Republic of Bulgaria）

| 维度 | 详情 |
|------|------|
| **现代疆域** | 南以罗多彼山脉和巴尔干山脉为界（与罗马接壤），北以多瑙河为界（与达契亚接壤），东临黑海；**占据多布罗加大部（含瓦尔纳、布尔加斯）** |
| **历史渊源** | 14世纪后中兴的罗马帝国将保加利亚向南夺取色雷斯和爱琴海的野心死死挡在巴尔干山脉以北；南方碰壁的保加利亚转向多瑙河下游及黑海西岸的多布罗加地区扩张 |
| **地缘姿态** | "背对东罗马、面向黑海与俄罗斯"——与达契亚、鞑靼人、沙俄在多布罗加长期拉锯后成功吞并 |

### 达契亚共和国（Republic of Dacia / 替代现实罗马尼亚）

| 维度 | 详情 |
|------|------|
| **现代疆域** | 多瑙河以北，喀尔巴阡山脉以东；仅拥有**康斯坦察及周边狭长的黑海海岸线** |
| **历史渊源** | 保加利亚的强势挤压使达契亚（瓦拉几亚与摩尔达维亚的结合体）失去对多布罗加的控制，成为半内陆国家 |
| **地缘依赖** | 近代极度依赖北方俄罗斯与西方哈布斯堡，以对抗南方保加利亚与罗马共和国 |

> **命名注**：本时间线中不存在"罗马尼亚"（Romania）这一国名——该词在本世界中指代罗马共和国自身（Romania = 罗马人的土地）。现实罗马尼亚对应的国家为**达契亚（Dacia）**。

### 塞尔维亚王国（Kingdom of Serbia）

| 维度 | 详情 |
|------|------|
| **现代疆域** | 占据现塞尔维亚大部（不含科索沃——科索沃归北阿尔巴尼亚），以及波黑东部（东正教区） |
| **地缘定位** | 罗马共和国在内陆的"小兄弟"，罗马向中欧投射影响力的"陆上跳板" |
| **经济纽带** | 两国通过**"东地中海-多瑙河经济走廊"**深度绑定 |
| **历史演进** | 1355年杜尚死后帝国碎裂；东罗马长期扶持东正教塞尔维亚大公作为抵御哈布斯堡南下的缓冲国；19世纪在东罗马暗中输血下完成统一 |

### 克罗地亚共和国（Republic of Croatia）

| 维度 | 详情 |
|------|------|
| **现代疆域** | 占据现克罗地亚大部及波黑西部（天主教区） |
| **地缘定位** | 受哈布斯堡/奥地利文化影响深远，是西方阵营（北约/欧盟）在巴尔干西北的桥头堡 |
| **与罗马摩擦** | 对罗马共和国控制的拉古萨-科托尔飞地如鲠在喉——切断克罗地亚南部海岸线连贯性；两国在亚得里亚海海洋权益与历史正统上常年外交摩擦 |
| **历史演进** | 没有奥斯曼威胁，未经历"军事边疆区"洗礼；作为匈牙利附属平稳融入哈布斯堡帝国；20世纪中叶独立 |

### 波斯尼亚共和国（Republic of Bosnia）

| 维度 | 详情 |
|------|------|
| **现代疆域** | 大致相当于现实中波黑 Federation 区域（不含塞尔维亚族区） |
| **宗教特色** | **波斯尼亚教会在夹缝中存活**——剥离"波格米勒派"二元论异端色彩，吸收东正教仪式和天主教组织架构，形成独特**"斯拉夫本土基督教派"** |
| **地缘定位** | 高度世俗化、奉行中立政策的**"巴尔干瑞士"**；罗马共和国与哈布斯堡/西方阵营之间的完美缓冲国 |
| **历史关键** | 没有奥斯曼征服 → 没有大规模伊斯兰化 → 波斯尼亚教会得以进行激进宗教改革 → 独特宗教认同阻止了被克罗地亚或塞尔维亚同化 |

### 阿尔巴尼亚共和国（Republic of Albania / 北移版）

| 维度 | 详情 |
|------|------|
| **现代疆域（整体北移）** | 占据现阿尔巴尼亚北部、科索沃地区以及黑山南部；**首都设在斯库台（Shkodër）或普里兹伦** |
| **宗教认同** | 以**天主教和山地部落文化**为主导（北部盖格人/Ghegs）；亲中欧/哈布斯堡 |
| **南部同化** | 阿尔巴尼亚中南部（托斯克人/Tosk 聚居区）信仰东正教，15-18世纪被罗马帝国深度文化同化——都拉斯、发罗拉完全"希腊-罗马化"，南部阿尔巴尼亚人融入"公民罗马主义"，自视为"讲伊利里亚方言的罗马公民" |
| **北部抵抗** | 北部盖格人在哈布斯堡和威尼斯暗中支持下坚决抵抗罗马同化；民族核心与政治中心被迫向北收缩至普罗克勒提耶（Prokletije）崎岖山地 |
| **民族情绪** | 痛恨南方"数典忘祖"的罗马化同胞；罗马共和国将南部视为不可分割的"伊利里亚大区" |

## 二、罗马共和国的巴尔干领地

### 色雷斯大区

**边境重镇：普罗夫迪夫（Philipopolis）**——罗马共和国北方边境城市，拱卫色雷斯平原的"北方门锁"。自中世纪起矗立着针对保加利亚与沙俄联军的庞大星形棱堡群。以北为保加利亚控制的巴尔干山脉隘口；以南为罗马繁华的色雷斯腹地，直通爱琴海。

### 伊利里亚大区

| 边境城市 | 角色 |
|----------|------|
| **都拉斯（Dyrrhachium）** | 亚得里亚海东岸核心港口，防御北方山地部落南下的海上堡垒 |
| **斯科普里（Skopje）** | 马其顿大区北部边境城市，抵御塞尔维亚与北阿尔巴尼亚民族主义渗透的"陆上防波堤"；驻扎机械化山地师 |

### 亚得里亚海飞地：拉古萨-科托尔大区

> "达尔马提亚罗马大区"——罗马共和国版图上最独特的拼图，像一把匕首从南方海岸向北突出，深深楔入亚得里亚海。

| 维度 | 详情 |
|------|------|
| **组成** | 拉古萨（Dubrovnik）+ 科托尔（Kotor）及南达尔马提亚沿岸 |
| **人口** | ~80万 |
| **语言** | 希腊语 + "达尔马提亚-罗马方言"（融合威尼斯语词汇） |
| **宗教** | 东正教与天主教和平共处——独特"亚得里亚海礼派" |
| **经济** | 地中海顶级度假胜地与免税金融中心 |

**归属变迁**：

| 时代 | 事件 |
|------|------|
| 中世纪至近代 | 拉古萨作为独立共和国在威尼斯、哈布斯堡和东罗马夹缝中靠外交生存 |
| 19世纪初 | 拿破仑摧毁拉古萨共和国 |
| 19世纪巴尔干分赃 | 东罗马以"保护东正教商路与历史遗产"为由出兵并合法吞并 |
| 一战/巴黎和会 | 英法为惩罚意大利贪婪，正式确认拉古萨为罗马帝国合法领土 |
| 二战 | 凭"亲德中立"免遭战火，成为地中海最大间谍与走私中心 |
| 现代 | 科托尔湾驻扎共和国最先进AIP常规潜艇与防空导弹阵地 |

**政治地位**：在共和国内部享有**高度自治权**，拥有自己的区旗（圣布莱斯与双头鹰结合），在君士坦丁堡议会拥有专属5个席位。

**周边关系**：
- **克罗地亚**：飞地切断其南部海岸线连贯性；克罗地亚修建跨海大桥绕过飞地领海
- **意大利**：科托尔的雷达站和反舰导弹使意大利东海岸（巴里、安科纳）完全暴露在罗马火力覆盖下——意大利在欧盟内对罗马常采取强硬态度的历史根源

## 三、大国关系

### 意大利：永恒的"谁是罗马"之争

| 维度 | 详情 |
|------|------|
| **核心矛盾** | 意大利民族主义者视东罗马为"窃取罗马名字的希腊伪物"；双方争夺"罗马"法统、达尔马提亚海岸及地中海霸权 |
| **一战** | 意大利加入协约国——英法《伦敦条约》许诺达尔马提亚、阿尔巴尼亚保护国甚至安塔利亚；1916年罗马倒戈后被迫与昨日之敌成为"盟友"，在阵营内疯狂"抢地盘" |
| **巴黎和会背刺** | 英法制衡意大利，将大部分达尔马提亚和阿尔巴尼亚控制权交给东罗马——比现实中深刻十倍的"残缺的胜利"（Vittoria mutilata） |
| **"希腊式的讹诈"** | 德国高层对罗马帝国一战外交的刻板印象——法纳尔政客在同盟国与协约国间反复横跳、待价而沽，利用海峡地缘优势勒索德国军费 |
| **法西斯主义** | 墨索里尼上台的核心纲领之一：**"打倒君士坦丁堡的伪物，确立台伯河罗马的唯一正统"**；二战中加入轴心国的深层原因——德国承诺支持夺取达尔马提亚并默许对东罗马发动"正统战争" |
| **现代** | 两国围绕"谁是真罗马"和亚得里亚海霸权的斗争仍是地中海世界最无解的地缘死结 |

### 奥地利/哈布斯堡：时战时和

| 维度 | 详情 |
|------|------|
| **历史模式** | 始终对巴尔干西北部（塞尔维亚、瓦拉几亚）抱有领土野心；与罗马时而联合遏制俄罗斯，时而暗中对抗 |
| **1878年分赃** | 奥匈吞并波斯尼亚，试图推行"超民族主义"同化斯拉夫人——因没有穆斯林制衡，只能残酷镇压东正教徒起义 |
| **灵活外交** | 罗马时而联合奥地利遏制俄罗斯在波兰的扩张，时而暗中支持匈牙利新教徒牵制维也纳 |
| **现代** | 欧盟框架内的合作伙伴，但历史记忆中的巴尔干争夺仍是暗流 |

### 俄罗斯/苏联：从"东正教盟友"到"东正教对手"

| 维度 | 详情 |
|------|------|
| **扩张受限** | 南方横亘着近代化且拥有强大海军的东罗马帝国，沙皇"十次俄土战争"南下黑海和巴尔干的成本高不可攀 |
| **转向方向** | 加速瓜分波兰-立陶宛联邦（西方）；目光投向高加索山脉以南的波斯残部与中亚草原（南方） |
| **"第三罗马"冷战** | 俄拜之间存在争夺"东正教共主"的冷战，但全面战争大幅减少 |
| **克里米亚** | 1812年罗马主动撤出克里米亚（"断臂求生"），换取沙俄承认博斯普鲁斯海峡为罗马"内水" |
| **现代** | 苏联解体后在黑海维持"冷和平"；罗马利用海峡通行法案在俄乌冲突中巧妙限制俄罗斯黑海舰队 |

### 克里米亚：战略性截肢

| 阶段 | 事件 |
|------|------|
| 14–16世纪 | 罗马重新加强克里米亚南岸控制，采取"以夷制夷"——向鞑靼汗国提供黄金和火器充当缓冲 |
| 17–18世纪 | 沙俄强势南下，罗马陷入致命战略两难——跨越黑海在远离本土的大草原与沙俄打消耗战毫无胜算 |
| **1812年《黑海密约》** | **"世纪大交易"**——罗马全面撤出克里米亚及黑海北岸所有据点；沙俄承认罗马对多瑙河下游、巴尔干山脉以南、高加索西岸的绝对控制权，并承认博斯普鲁斯海峡为罗马"内水" |
| 1853–1856 克里米亚战争 | 罗马选择"武装中立"——乐见英法联军在塞瓦斯托波尔与沙俄互相放血，封锁海峡防止沙俄黑海舰队突围 |
| 现代 | 罗马明确表示**克里米亚属于乌俄历史争议，不持立场，绝不寻求重返黑海北岸** |

**地缘逻辑**：黑海北岸是"陆权国家的绞肉机"，黑海南岸和海峡才是"海权国家的生命线"。只要控制博斯普鲁斯海峡和巴统军港，黑海北岸无论谁当家作主都必须看君士坦丁堡脸色。

## 四、21世纪巴尔干"新三国演义"

| 阵营 | 成员 | 特征 |
|------|------|------|
| **紫色阵营（罗马霸权圈）** | 罗马共和国（核心）+ 色雷斯、马其顿、伊利里亚（阿中南部）+ 盟友塞尔维亚 | 掌握爱琴海与亚得里亚海南部绝对霸权 |
| **黑色阵营（哈布斯堡-天主教圈）** | 克罗地亚、北阿尔巴尼亚、波斯尼亚西部 | 背后是奥地利/中欧势力支持，试图将罗马势力阻挡在达尔马提亚山脉以北 |
| **灰色阵营（黑海-斯拉夫圈）** | 保加利亚、达契亚 | 在俄罗斯与罗马的夹缝中维持多瑙河-黑海区域平衡 |

**核心分裂线**：东正教罗马 / 天主教哈布斯堡-意大利 / 斯拉夫保加利亚-俄罗斯
