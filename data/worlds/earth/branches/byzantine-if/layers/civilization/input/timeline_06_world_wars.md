# 世界大战：皇冠落地与实用主义生存（1910–1950）

1910至1950年是帝国历史上最具决定性的四十年。一战中加入同盟国阵营，以"希腊讹诈"策略向德国索取让步，但1916年南方阿拉伯行省叛变迫使帝国倒戈投向协约国。英法以"民族自决"为武器保护叛军，帝国止步大马士革一线。1919年巴黎和会的"世纪背刺"摧毁了帕莱奥洛根王朝的合法性，1922年君主制崩溃，第二共和国宣告成立。大萧条引爆四派内战，右翼军政府于1936年惨胜后奉行"亲德中立"的武装生存哲学，在二战中拒做炮灰、见风使舵，最终以近乎完整的领土保全走出战争。

→ 帝国主义时代的结构性矛盾见 `timeline_05_imperialism.md`

## 一、第一次世界大战（1914-1918）

### 战前外交孤立与参战抉择

帝国在帝国主义时代末期面临同盟国（德、奥匈）与协约国（英、法、俄）的致命拉扯。基于以下考量加入同盟国：

- 苏伊士运河共管导致英拜关系降至冰点
- 与奥匈帝国在巴尔干达成妥协（→ 见 `timeline_05_imperialism.md`）
- 对俄罗斯在巴尔干和高加索的长期地缘竞争
- 德意志第二帝国的工业与军事合作吸引力

### 1914-1916：同盟国阵营中的"希腊讹诈"

帝国在同盟国内部采取"以退出相要挟"的策略，持续向德国索取让步：

| 讹诈手段 | 获得回报 |
|---------|---------|
| 威胁单独媾和 | 德国提供工业设备与军事技术 |
| 拖延出兵时间 | 获得经济贷款与关税优惠 |
| 要求修改战后分配方案 | 德国承诺中东势力范围让步 |

此策略被德国军方称为"希腊讹诈"（Greek Blackmail），短期内为帝国积累了 substantial 物资储备，但也加剧了德拜之间的信任赤字。

### 1916：南疆叛变与帝国倒戈

**帝国主力的战略困境**——1916年帝国陆军主力被钉在两条残酷战线上：

| 战线 | 方位 | 对手 | 态势 |
|------|------|------|------|
| 摩尔达维亚-多布罗加前线 | 西北 | 沙俄—达契亚联军 | 喀尔巴阡山脉以东的冰雪拉锯战 |
| 高加索-埃尔祖鲁姆前线 | 东方 | 沙俄高加索集团军 | 安纳托利亚东部堑壕战 |

> **注**：本时间线中瓦拉几亚与摩尔达维亚地区的独立国家使用古典名称**"达契亚王国"（Kingdom of Dacia）**，以避免与帝国自称的"罗马"（Romania）产生法理冲突。达契亚于1916年夏加入协约国，试图夺取多布罗加地区。

**南疆叛变**——主力被沙俄和达契亚拖住，南方叙利亚与巴勒斯坦大区兵力极度空虚，仅留守二线阿拉伯裔地方部队和宪兵。1916年秋，在英法情报机构的黄金与"阿拉伯独立"承诺诱惑下，大马士革与耶路撒冷的阿拉伯裔总督及军团宣布起义：

- 切断帝国通往苏伊士运河的陆路补给线
- 直接威胁亚历山大港侧翼安全
- 形成北（沙俄+达契亚）、东（高加索）、南（叛军+英军）三面受敌的亡国级危机

**倒戈决策**——为保住核心本土（色雷斯与小亚细亚），帝国高层于1916年底背刺同盟国，向协约国乞和并倒戈。

### 1917-1918：英法的"法理绞杀"与戴镣铐的平叛

英法欢迎帝国倒戈（封锁黑海海峡、减轻中东压力），但绝不容许战后出现独霸中东的"超级罗马"：

**英法的马基雅维利主义**：

1. **"保护圣地"武装干涉**：英军以保护耶路撒冷宗教圣地为由，从西奈半岛北上进驻巴勒斯坦，成为叛军的实质保护伞
2. **"民族自决"法理陷阱**：将叛军包装为"反抗旧帝国压迫的自由战士"，使帝国在法理上无法对受保护的"独立实体"发动全面战争

**平叛战争的诡异格局**：

- 帝国多线作战，能投入南疆的兵力严重不足
- 英法表面呼吁"停火与和谈"，暗中通过埃及向西奈叛军提供武器（李-恩菲尔德步枪、野战火炮、早期装甲车）
- 英军顾问着便服指挥叛军进行沙漠游击战
- 帝国军队凭更优战术素养缓慢南推，扫清北叙利亚叛军据点
- 逼近大马士革时遭遇英军"维和部队"与叛军联合防线
- 为避免与英军直接武装冲突，帝国指挥官被迫停止进攻

**1918年停战时帝国实际控制线**：大马士革以北。

### 1919年巴黎和会："世纪背刺"

| 英法操作 | 帝国损失 |
|---------|---------|
| "大马士革以南已在战争中事实独立" | 拒绝帝国恢复战前边界 |
| 巴勒斯坦（含耶路撒冷）→ 英国委任统治 | 丧失圣城 |
| 南叙利亚（黎巴嫩及叙利亚南部）→ 法国委任统治 | 丧失南疆 |
| 亚历山大港：帝国直属海外大区，无大规模叛乱 | **得以保留** |

> **亚历山大港幸存的原因**：自19世纪中叶起即为帝国直属海外大区（→ 见 `timeline_04_revolution.md`），居民多为拥有罗马国籍的希腊/拉丁/阿拉伯混血后裔，且为帝国地中海舰队母港。英法无法以"民族自决"为由剥夺，且需帝国海军维持战后地中海秩序。

### "残缺的胜利"（Vittoria Mutilata）

帝国作为战胜国，不仅未收复圣城，反而眼见叛变领土被盟友合法瓜分。前线将士在摩尔达维亚和高加索流尽鲜血，却在谈判桌上被英法用"民族自决"剥夺圣城。

- **右翼狂怒**：退伍军人和民族主义者痛斥"在战场上赢了，在谈判桌上输了"
- **左翼觉醒**：工人和知识分子认为战争不过是"帝国主义列强分赃的闹剧"
- **王朝合法性彻底崩溃** → 直接催生1920年代的"紫室风暴"

## 二、君主制崩溃与共和国诞生（1919-1925）

- 1919-1922年间国内政治极化加剧，右翼军人团体与左翼工会同时冲击政府
- 1922年帕莱奥洛根王朝末代皇帝退位流亡，**第二罗马共和国**宣告成立
- 初期建立议会民主制，但社会裂痕深远

## 三、大萧条与"四派乱斗"（1925-1932）

1929年华尔街股灾引发全球大萧条，共和国经济命脉遭毁灭性打击：

| 经济指标 | 冲击 |
|---------|------|
| 亚历山大棉花出口 | 暴跌 |
| 苏伊士运河红利 | 被英法资本吸血 |
| 国内通胀 | 恶性通胀，中产阶级破产 |
| 失业率 | 飙升至30% |

**四大政治阵营**：

| 阵营 | 别称 | 社会基础 | 核心主张 |
|------|------|---------|---------|
| 主战派 | "军团派" | 一战退伍军人、青年军官、极端民族主义者 | 极权政府、收复耶路撒冷、打碎凡尔赛体系 |
| 革命派 | 左翼/苏维埃倾向 | 破产农民、城市工人、激进知识分子 | 没收大地主和教会土地、建立"巴尔干-小亚细亚苏维埃联邦" |
| 主和派 | 自由派/议会派 | 工商业资产阶级、法纳尔知识分子 | 维持现状、依靠英法贷款、捍卫共和宪政 |
| 保皇派 | — | 东正教高层、旧贵族、安纳托利亚地主 | 迎回帕莱奥洛根皇室、恢复君主立宪制 |

## 四、"第二次紫室内战"与外国干涉（1932-1936）

1932年冬，主和派政府强行削减军费并镇压左翼罢工，引爆全面内战：

- **革命派**：在塞萨洛尼基和士麦那发动武装起义
- **主战派**：在君士坦丁堡发动"向紫室进军"政变
- **保皇派**：在地方起事

### 外部势力干涉

| 干涉方 | 支持对象 | 援助形式 |
|--------|---------|---------|
| 苏联 | 革命派 | 有限军援、共产国际志愿军（规模不及史实西班牙内战） |
| 纳粹德国 | 主战派 | "条顿鹰大队"、斯图卡轰炸机、装甲车 |
| 法西斯意大利 | 主战派 | 军事援助（试图借机控制爱琴海） |
| 英法 | 主和派 | 道义同情、有限人道主义物资（奉行绥靖政策，拒绝军事干预） |
| — | 保皇派 | 缺乏现代军事组织，被主战派吞并 |

**内战后果**：

- 四年堑壕战、山地游击战和城市轰炸
- 君士坦丁堡圣索菲亚大教堂在轰炸中受损
- 东地中海大铁路被炸毁
- 国库空虚，军队疲惫不堪
- **1936年主战派惨胜**：建立以"第一公民"为首的**右翼威权军政府**（参照佛朗哥/梅塔克萨斯模式）

## 五、威权重建与"亲德中立"（1936-1939）

军政府上台后展现冷酷的地缘理性：

| 策略 | 具体操作 |
|------|---------|
| 意识形态靠拢 | 反共、威权主义、古典帝国主义与纳粹德国有共鸣 |
| 拒绝成为附庸 | 以"内战创伤未愈"为由拒绝加入轴心国军事同盟 |
| 确立"武装中立" | 1938年签订《柏林-君士坦丁堡友好与互不侵犯条约》 |
| 经济交换 | 向德国出口铬矿、农产品、石油，换取工业设备 |
| **底线** | **坚决拒绝加入军事同盟** |

## 六、二战中的"罗马堡垒"（1939-1945）

1939年二战爆发，共和国宣布中立，成为地中海-中东地区的"风暴眼"。

### 关键决策

| 事件 | 时间 | 共和国应对 | 结果 |
|------|------|-----------|------|
| 意大利入侵企图 | 1940 | 动员80万常备军边境陈兵 | 希特勒训斥墨索里尼，入侵计划取消 |
| 巴巴罗萨行动 | 1941 | 拒绝出兵，仅提供象征性后勤过境权 | 军队避免斯大林格勒式覆灭 |
| 谍报活动 | 1939-1945 | 君士坦丁堡成为全球最大间谍之都 | 同盟国与轴心国激烈暗战与情报交易 |
| 亚历山大港 | 1939-1945 | 交战国交换战俘、购买战略物资的"白手套"港口 | 赚取海量黄金和外汇 |

### 1944年"八月清洗"与跳船

- 1944年夏，盟军诺曼底登陆、苏联红军逼近巴尔干
- 军政府果断逮捕国内亲德极端派
- 宣布对德宣战
- 向英美开放部分领空和海军基地
- 勉强挤入"战胜国"行列

## 七、战后清算与领土保全（1945-1950）

### 苏联的止步

- 苏联红军席卷巴尔干北部，但在色雷斯和马其顿边境止步
- 斯大林判断：共和国拥有上百万未受损百战之军，民族主义情绪极其狂热（类似铁托的南斯拉夫但更亲西方），强行吞并必陷泥潭
- 苏联默认共和国为东西方"武装中立缓冲国"

### 英美的拉拢

- 英美需强大地区盟友防止苏联控制东地中海和苏伊士运河
- 共和国凭"反戈一击"和完整保留的军事工业体系获得**马歇尔计划**援助

### 1947年《巴黎和平条约》

共和国作为"名义上的战胜国/中立国"，**国界线相比1939年战前几乎无任何变化**：

- 色雷斯、马其顿、希腊半岛 ✓
- 小亚细亚中西部 ✓
- 叙利亚（至大马士革一线） ✓
- 亚历山大港 ✓

未丢失一寸核心领土，未遭敌军全面占领。

## 八、1950年的共和国

| 维度 | 状态 |
|------|------|
| 政治 | 右翼军政府逐渐还政于民→保守右翼民主政府；军队和情报机构仍为"深层政府" |
| 经济 | 未受战火摧残的工业基础+马歇尔计划→战后"经济奇迹" |
| 城市 | 君士坦丁堡和亚历山大港复兴为地中海金融与航运中心 |
| 地缘 | **"东地中海不结盟强权"**——非北约正式成员、非华约附庸 |
| 军事 | 强大陆军+扼守黑海-地中海咽喉的海军，在美苏间左右逢源 |

→ 冷战时期的民主化转型与中东博弈见 `timeline_07_modern.md`
