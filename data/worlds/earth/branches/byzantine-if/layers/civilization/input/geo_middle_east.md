# 中东与高加索地缘

罗马共和国的东部边界不是一条简单的国境线，而是一道深深刻在欧亚大陆板块交界处的**"文明断层线"**。从巴统的军港到凡湖的界碑，从阿勒颇的废墟到耶路撒冷的圣墓，罗马用现代化装甲师、无人机与资本，维系着一个"新拜占庭式"的朝贡体系。

→ 总览见 `_overview.md`；欧洲地缘见 `geo_europe.md`；宗教与文化见 `religion.md`

## 一、奥斯曼的灭亡与安纳托利亚东部边疆

### 奥斯曼的永久覆灭（1402年）

1402年帖木儿安卡拉战役彻底歼灭奥斯曼军队。在本时间线中，拜占庭未因内战而衰弱，强大海军严密封锁达达尼尔海峡，导致奥斯曼无法从欧洲获取资源、国力受限，被帖木儿摧毁后未能像真实历史那样在"大空位期"后复兴。**安纳托利亚重回突厥诸贝伊国割据的碎裂状态。**

拜占庭趁势渡海，兵不血刃地收复小亚细亚西北部核心故地——尼西亚（Nicaea）、普鲁萨（Prusa）、士麦那（Smyrna），将国境线推回安纳托利亚高原边缘。

### 安纳托利亚东部边界

**边界走向**：迪亚巴克尔 → 托罗斯山脉 → 幼发拉底河上游

| 区域 | 归属 | 说明 |
|------|------|------|
| 迪亚巴克尔 | 罗马 | 水源与兵源地，高度军事化 |
| 埃尔祖鲁姆、穆什、比特利斯 | 罗马 | 安纳托利亚东部堡垒群——**"东方马奇诺防线"** |
| 卡尔斯 | 亚美尼亚 | 亚美尼亚西北门户 |
| 阿勒山（Agri） | 两国界山 | 亚美尼亚圣山，文化主权争夺焦点 |
| 凡城（东岸） | 亚美尼亚 | 凡湖分治 |
| 凡湖（西岸/南岸） | 罗马 | 现代农业、重化工、环湖度假村 |

→ 亚美尼亚关系详见下文"高加索诸国"

## 二、高加索诸国

高加索在本时间线中是**罗马（东正教/希腊-罗马文化）、沙俄/苏联（东正教/斯拉夫文化）与波斯（什叶派伊斯兰）**三大帝国的直接碰撞区。

### 历史变迁概览

| 时期 | 格局 |
|------|------|
| **1341–1500** | 帖木儿铁蹄与黑羊/白羊王朝游牧征服将格鲁吉亚和亚美尼亚打成碎片；拜占庭退守特拉布宗与黑海沿岸 |
| **1500–1800** | 萨法维波斯崛起，将东亚美尼亚与阿塞拜疆纳入什叶派版图；拜占庭以"安纳托利亚东部堡垒群"挡住波斯西进 |
| **1800–1914** | 沙俄吞并格鲁吉亚（1801）与东亚美尼亚（1828）；高加索形成"北俄、南波、西罗"三足鼎立 |

### 近代血腥事件：高加索-黎凡特派系清洗（1890s–1910s）

> **替代亚美尼亚大屠杀的设定**

由于没有奥斯曼帝国的系统性国家组织屠杀，本时间线的悲剧表现为**前线战区的无政府状态与代理人仇杀**（Caucasus-Levant Sectarian Cleansing）：

- 沙俄暗中武装亚美尼亚与格鲁吉亚民族主义民兵
- 波斯与英国资助库尔德与阿塞拜疆突厥语穆斯林武装
- 双方在安纳托利亚东部与高加索南部展开惨烈的种族清洗与互相驱逐
- **去中心化暴力**——非中央政权主导，而是底层仇恨与帝国特工煽动的互屠
- 数百万基督徒与穆斯林在风雪中流离失所，凡湖与阿勒山周边沦为"无人区"

**历史罗生门**：
- **亚美尼亚叙事**：指责罗马"安纳托利亚东部军团"故意拖延救援，借穆斯林之手削弱亚美尼亚在凡湖以西的势力
- **罗马叙事**：反驳是亚美尼亚民兵率先劫掠罗马东正教村庄，试图将卡尔斯与阿勒山周边"亚美尼亚化"

### 格鲁吉亚共和国

| 维度 | 详情 |
|------|------|
| **领土** | 第比利斯、库塔伊西；**失去巴统（归罗马）** |
| **地缘定位** | 坚定的亲罗/亲西方阵营；北约（或地中海联盟）雷达站密布其边境 |
| **巴统之刺** | 罗马黑海第一舰队母港，部署防空导弹阵列与AIP常规潜艇；罗马视巴统为不可触碰的逆鳞 |

**罗马-格鲁吉亚"带刺血盟"**：

巴统剥离历史：
1. **15世纪"保护性接管"**：格鲁吉亚王国解体为三个碎片，罗马皇帝以"东正教保护者与宗主"身份派舰队进驻巴统，设为"拉齐卡军管区"——对当时格鲁吉亚王公而言是用一座港口换取罗马军团抵御异教徒的"划算交易"
2. **19世纪"帝国分赃"**：1820年代《黑海海峡与高加索协定》——沙俄承认罗马对巴统及向南延伸至特拉布宗沿海地带的绝对主权；罗马默认沙俄吞并格鲁吉亚内陆
   - 格鲁吉亚民族主义者眼中：罗马将格鲁吉亚内陆"卖"给了俄罗斯
   - 罗马法理叙事：巴统自15世纪起就由君士坦丁堡直接管辖的"皇室领地"

同盟的冷酷逻辑：
- 格鲁吉亚**根本无法脱离罗马羽翼**——北方俄罗斯/苏联威胁、南方波斯/阿塞拜疆压力、经济命脉依赖巴统港与特拉布宗港
- 格鲁吉亚政党"**口头收复、现实妥协**"——竞选时高呼"巴统是格鲁吉亚不可分割的灵魂故土"，执政时小心翼翼避免触碰罗马军事底线
- 罗马对格鲁吉亚公民实行"**有条件的免签**"——签署"承认巴统为罗马历史领土"的免责声明即可自由进入罗马本土打工

**能源博弈**：BTC管道（巴库-第比利斯-巴统/杰伊汉）为欧洲能源大动脉；格鲁吉亚试图利用过境国地位提高过路费，罗马以"切断电网"威慑——双方在密室中达成"资源换安全"妥协。

### 亚美尼亚共和国

| 维度 | 详情 |
|------|------|
| **领土** | 埃里温、塞凡湖、卡尔斯、阿勒山、凡城（东岸）；比现实多一部分领土 |
| **地缘定位** | **"高加索刺猬"**——在罗马的资本、俄罗斯的能源与波斯的贸易之间走钢丝 |
| **凡湖** | 两国分治的界湖；气候变化导致水位下降引发水资源争端，多次"水炮船对峙" |

**罗马-亚美尼亚千年神学宿怨**：

| 时期 | 关系 |
|------|------|
| 451年迦克墩公会议后 | 亚美尼亚使徒教会坚持**"合性论"（Miaphysitism）**，拒绝拜占庭"神人二性"说——被视为"比穆斯林更顽固的异端" |
| 11世纪 | 拜占庭强行吞并亚美尼亚巴格拉提德王朝（1045年吞并阿尼），废除当地贵族武装 → 塞尔柱突厥来袭时亚美尼亚冷眼旁观 → 1071年曼齐刻尔特惨败 |
| 13–14世纪 | 面对突厥共同威胁展开深度联姻；亚美尼亚军事贵族流入君士坦丁堡成为高级将领 |
| 1890s–1910s | 派系清洗中既是并肩战友又发生底层互屠——"历史罗生门" |
| **1990s** | 签署历史性**《凡湖宣言》**——搁置1500年神学争议，确立"东方古老教会共融"原则 |

**现代关系**："没有爱情，只有生存"的地缘婚姻——

- **防线共享**：罗马"安纳托利亚东部堡垒群"与亚美尼亚"阿拉拉特防线"互为犄角；罗马无偿提供"拜占庭-B"系列自杀式无人机与防空雷达数据链
- **俄罗斯杠杆**：亚美尼亚"亲俄派"试图引入俄军驻扎时，罗马以"关闭特拉布宗-卡尔斯-埃里温铁路"威慑——亚美尼亚经济一周内面临停摆
- **阿勒山文化暗战**：亚美尼亚将阿勒山印在国徽、货币和白兰地酒标上；罗马在西侧修建"诺亚方舟国家考古公园"，宣扬阿勒山是"全人类共同起源"

### 阿塞拜疆共和国

| 维度 | 详情 |
|------|------|
| **领土** | 巴库、占贾 |
| **地缘特征** | 由于没有奥斯曼的泛突厥主义背书，20世纪更多受波斯（什叶派）与俄罗斯影响 |
| **政体** | 高度威权的石油国家 |
| **与亚美尼亚矛盾** | 纳卡（纳戈尔诺-卡拉巴赫）及赞格祖尔走廊问题常年对峙 |

## 三、中东诸国

### 叙利亚（含黎巴嫩领土）：没有缓冲的"火药桶"

> **关键设定**：本时间线**不存在独立的黎巴嫩**——贝鲁特、的黎波里等黎巴嫩领土整体划入叙利亚。

| 维度 | 详情 |
|------|------|
| **现代疆域** | 拥有塔尔图斯、贝鲁特、的黎波里、大马士革；**不含**拉卡-霍姆斯一线以北（归罗马控制） |
| **罗马-叙利亚边界** | 迪亚巴克尔 → 拉卡 → 阿勒颇 → 霍姆斯 → 拉塔基亚 |
| **黎巴嫩消失的后果** | 马龙派、逊尼派、阿拉维派、德鲁兹派失去外部缓冲，在缺乏强力世俗政府的国家内陷入长达百年的教派仇杀 |
| **三次内战** | 自1950年代以来爆发三次惨烈内战 |
| **罗马的角色** | "北方宗主"——常年以"保护东正教/基督徒侨民"为由对叙利亚北部（阿勒颇-霍姆斯一线）实施军事占领与经济吸血 |

**历史变迁**：1516年被马穆鲁克控制 → 成为波斯与拜占庭争夺战场 → 19世纪中叶拜占庭借埃及大乱夺取叙利亚北部 → 一战中南叙利亚在英法支持下独立。

### 伊拉克（两河）：波斯的后花园

| 维度 | 详情 |
|------|------|
| **领土** | 巴士拉、巴格达、摩苏尔——基本维持现实版图 |
| **历史** | 长期受波斯（萨法维/卡扎尔）控制；19世纪末拜占庭试图通过"巴格达铁路支线"渗透被英波联手阻击；一战后成为英国委任统治地 |
| **地缘** | 北方面临罗马地缘高压，外交长期倒向波斯（伊朗）或俄罗斯——形成**"巴格达-德黑兰-莫斯科"隐性同盟** |

### 巴勒斯坦/以色列与耶路撒冷

| 实体 | 详情 |
|------|------|
| **耶路撒冷** | 联合国/四方委员会共管（联合国、罗马/东正教、梵蒂冈/天主教、伊斯兰瓦克夫）；罗马在圣墓教堂周边拥有事实"治外法权区" |
| **以色列** | 建国过程更加曲折（罗马作为东正教大国强力干涉——既同情犹太人苦难又须保护阿拉伯裔东正教徒）；领土更小，与周边形成复杂"武装和平" |
| **巴勒斯坦** | 与以色列共存于更紧张的地缘环境中 |

→ 耶路撒冷详情见 `religion.md`

### 外约旦（哈希姆王国）

英国传统盟友，夹缝中生存的温和缓冲国——罗马共和国与阿拉伯世界之间罕见的"温和缓冲区"。

### 阿拉伯半岛

罗马共和国控制亚历山大港与红海北端，对汉志（麦加、麦地那）朝觐航线拥有绝对控制权。沙特家族统一半岛过程中不得不向君士坦丁堡缴纳巨额"过路费"或给予石油开采特权。现代沙特在华盛顿、伦敦与君士坦丁堡之间维持微妙平衡。

## 四、关键地缘主题

### 罗马作为"中东守门人"

| 控制节点 | 功能 |
|----------|------|
| 博斯普鲁斯海峡 | 黑海唯一出口，扼俄/乌命脉 |
| 巴统军港 | 黑海第一舰队母港，威慑高加索沿岸 |
| 安纳托利亚东部堡垒群 | "东方马奇诺防线"，拱卫小亚细亚腹地 |
| 叙利亚北部（阿勒颇-霍姆斯） | 军事占领区，经济吸血 |
| 亚历山大港飞地 | 扼守苏伊士运河，红海-地中海枢纽 |
| 塞浦路斯 | 东地中海天然气开采中心，情报灰色枢纽 |

### "新拜占庭式"朝贡体系

罗马不再追求直接吞并所有土地，而是通过**控制水源、铁路、港口与宗教圣地**，让高加索与中东诸国在敬畏与仇恨中围绕双头鹰阴影旋转：

- 格鲁吉亚：依赖巴统港与罗马电网
- 亚美尼亚：依赖特拉布宗-卡尔斯铁路
- 叙利亚：北部被军事占领
- 沙特：红海航线被控制
- 凡湖：水资源分配权
