# 拜占庭架空历史分支：第二罗马共和国

## 核心前提

**1341年，两约翰内战以坎塔库泽努斯的迅速胜利告终**，拜占庭帝国避免了真实历史中毁灭性的七年内战（1341-1347），没有在1453年灭亡，而是延续至今，蜕变为现代的**第二罗马共和国**（Second Roman Republic）。

## 关键转折点

- **1341** — 两约翰内战迅速结束，坎塔库泽努斯兵不血刃入主君士坦丁堡
- **1402** — 帖木儿在安卡拉彻底摧毁奥斯曼势力，罗马趁机光复小亚细亚腹地
- **1522** — 巴西殖民地（紫木之地/Porphyria）建立
- **1571** — 大分裂/紫室之劫：君士坦丁堡大地震后遭西班牙-威尼斯联军攻陷，帝国陷入分裂
- **1645** — 光复：拉斯卡里斯-科穆宁流亡政权从塞萨洛尼基打回君士坦丁堡
- **1799** — 第一共和国建立：法国大革命思潮冲击，末代皇帝被推翻，君主制首次法理中断
- **1812** — 克里米亚大交易：罗马主动放弃黑海北岸，换取博斯普鲁斯海峡"内水"地位
- **1824** — 巴西殖民地独立
- **1848** — 欧洲之春革命，公民军团在阿德里安堡击退沙俄干涉
- **1880s** — 殖民扩张：北非（利比亚）、东非（厄立特里亚）、远东租界
- **1916** — 一战倒戈：12月11日罗马背刺同盟国，转投协约国
- **1919** — 巴黎和会背刺：英法剥夺全部海外殖民地，失去耶路撒冷与南叙利亚
- **1922** — 第二共和国：紫室风暴，末代皇帝流亡伦敦，延续近两千年的君主制终结
- **1932-1936** — 内战：左右翼激烈冲突
- **1944** — 二战战略转折，罗马成为东地中海关键力量
