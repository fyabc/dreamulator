# 海外领地与殖民史

罗马共和国的海疆变迁是一部"从爱琴海内湖走向全球大洋，最终在帝国主义反噬中收缩为东地中海堡垒"的七百年史诗。1400–1500年肃清爱琴海拉丁人残余确立内湖霸权；1880–1910年在帝国主义狂飙中将旗帜插向北非、东非与远东；1919年巴黎和会背刺后彻底剥离无法防守的远洋殖民地，蜕变为极度务实的"区域执剑人"。

→ 总览见 `_overview.md`；亚历山大港详情见 `religion.md`；欧洲飞地见 `geo_europe.md`

## 一、紫木共和国（Republic of Porphyria / 前"新罗马尼亚" / 对应现实巴西）

### 国名起源

拜占庭探险家发现当地红木提取的染料在地中海阳光下氧化后呈现深邃**紫红色**。在罗马文化中，**紫色（Porphyra）是皇权与神圣的绝对象征**（如"紫室贵族"Porphyrogennetos）。皇室与教会将其视为"上帝赐予新罗马的皇权之树"，故命名为 **Terra Porphyria（紫木之地）**。独立后定名**波尔菲里亚共和国（Republic of Porphyria）**，中文常译"紫木共和国"或"紫袍共和国"。

### 七百年史诗

| 时期 | 名称 | 关键事件 |
|------|------|----------|
| **1522–1570** | 紫木黄金时代 | 拜占庭凭借重商主义传统在沿海建立以**新君士坦丁堡（Nea Konstantinoupolis，今萨尔瓦多）**为核心的贸易城邦网络；垄断"紫木染料"贸易，引进塞浦路斯甘蔗种植技术——"蔗糖与烟草心脏" |
| **1571–1640** | 海外罗马 | 1571年"紫室之劫"后——亲王带皇室血脉、残存远洋舰队、国库黄金、法纳尔学者与工匠跨大西洋逃亡至紫木之地；新君士坦丁堡成为东正教与古典罗马法统的"热带避难所"；吸收逃亡新教徒、犹太商人、非洲奴隶——高度商业化、宽容的**"克里奥尔罗马文化"**；英法暗中输血，1590年代南大西洋海战重创西葡联合舰队 |
| **1640–1790** | 热带雅典 | 本土光复后试图重新确立宗主权；1692年《萨尔瓦多法案》升格为**"共主邦联"内平等王国**；18世纪启蒙运动——新君士坦丁堡大学成为南美"热带雅典"；诞生**"泛美罗马主义"**——真正的罗马精神在热带雨林中获新生 |
| **1815–1824** | 独立战争 | 受美国独立与法国大革命影响，克里奥尔精英拒绝缴纳"帝国防卫税"；本土深陷巴尔干民族起义与财政枯竭无力跨洋镇压；1824年英国调停下《新君士坦丁堡和约》承认独立——保留"最惠国待遇"和东正教文化纽带 |
| **1824–1910** | 南美霸主 | 击败拉普拉塔（阿根廷前身），吞并乌拉圭全境确立**南美第一强国**地位；咖啡与橡胶繁荣引发移民狂潮——巴尔干失业希腊人、黎凡特阿拉伯人、意大利南部贫民涌入 |
| **一战** | 中立 | 向交战国出口农产品与矿产，完成早期工业化积累 |
| **二战** | 世纪和解 | 1942年因德国U型潜艇无限制击沉商船对轴心国宣战；**"紫木远征军"**前往北非与意大利本土与罗马军队并肩作战痛击意大利法西斯 |
| **1944** | 名场面 | 紫木远征军与罗马军队在罗马城外会师——紫木士兵将南十字星与双头鹰旗帜披在古罗马斗兽场残垣上，宣告"两个罗马"在反法西斯战火中的**世纪和解** |
| **冷战** | 摇摆 | 在亲美军政府与左翼民粹间剧烈摇摆，但始终是罗马共和国在南美最核心战略锚点 |
| **现代** | 南十字星 | **金砖国家（BRICS）**重要成员；南半球唯一拥有核潜艇（与罗马联合研发）和完整航空航天工业的国家 |

### 现代文化奇观

新君士坦丁堡狂欢节：人们穿着改良版拜占庭丝绸与古典托加，在东正教圣咏与桑巴鼓点交织中游行——**"罗马文明的南十字星"**。

## 二、殖民扩张对全球秩序的冲击

### 托尔德西拉斯条约的瓦解

东正教在1054年大分裂后**根本不承认罗马教皇的世俗仲裁权**，1494年西葡瓜分世界的《托尔德西拉斯条约》在君士坦丁堡眼中不过是"异端神棍的废纸"。拜占庭强行挤占南美东海岸，直接导致伊比利亚双雄势力范围剧烈偏移：

| 国家 | 偏移方向 | 后果 |
|------|----------|------|
| **葡萄牙** | 南向狂奔 | 被迫提前向西南扩张至**拉普拉塔河流域**（今乌拉圭、阿根廷）及亚马逊河口；与西班牙在南美南部冲突提前爆发且极其惨烈 |
| **西班牙** | 北方堡垒化 | 大量资源投入**加勒比海、哥伦比亚、委内瑞拉**防御——南美北部被打造为高度军事化的"白银堡垒"，经济开发滞后但军事要塞群密集 |
| **英法荷** | 浑水摸鱼 | 乐见天主教双雄与东正教拜占庭在南美"三国演义"；英国私掠船、法国胡格诺派、荷兰商人在**圭亚那地区和加勒比海岛链**建立大量走私据点——殖民归属比现实更加碎片化 |

## 三、亚历山大港飞地

| 维度 | 详情 |
|------|------|
| **获取** | 1840年代趁埃及大混乱，蒸汽铁甲舰队炮轰港口，以"保护东正教侨民与历史遗产"为由强行吞并 |
| **繁荣** | 苏伊士运河开通后爆炸式增长——帝国"非洲明珠"、棉花贸易中心、东地中海最强海军母港 |
| **二战** | "间谍之都"——因共和国"武装中立"/"亲德中立"成为交战国间最奇妙"缓冲阀"；同盟国与轴心国间谍在同一咖啡馆交换情报 |
| **现代** | **"赛博朋克式直布罗陀"**——城外埃及民族主义高墙，城内霓虹闪烁金融中心；居民拥有共和国护照，享受高福利与免签欧洲 |
| **争议** | 埃及持续争议主权；EMGF天然气论坛成员 |

## 四、东地中海岛屿

爱琴海与东地中海的诸多岛屿在13–14世纪被威尼斯、热那亚及十字军残余势力割据。1400–1500年间，中兴的罗马帝国凭借复兴海军，通过赎买、联姻与武力逐步驱逐"拉丁人僭主"，重建爱琴海绝对霸权。

### 克里特岛（Crete）

| 维度 | 详情 |
|------|------|
| **获取** | 1485年——帝国暗中资助希腊裔东正教徒"圣提多大起义"，威尼斯财政被拖垮后签订《干地亚条约》将克里特"卖"给罗马 |
| **1571年劫难** | "紫室之劫"中西班牙-威尼斯联军短暂夺回作为反攻跳板；1640年代光复战争中成为流亡政权最重要海军基地与粮仓 |
| **现代** | 南部海军枢纽与农业/旅游大岛；**苏达湾（Souda Bay）**部署最先进核潜艇/AIP潜艇基地，与亚历山大港飞地隔海构成控制东地中海的"双头钳" |

### 塞浦路斯岛（Cyprus）

| 维度 | 详情 |
|------|------|
| **获取** | 1489年——末代女王卡特琳娜·科纳罗在威尼斯逼迫下企图退位；罗马皇帝以"保护东正教臣民与古老拜占庭法统"为由抢先派舰队进驻法马古斯塔；通过**"紫室联姻"**与外交威慑和平接管 |
| **1571年** | 距君堡较远，未受战乱波及 |
| **现代** | 共和国最富裕海外大区之一；东地中海天然气开采（EMGF）核心指挥中心；与中东诸国情报交易的"灰色枢纽" |

### 基克拉泽斯群岛/纳克索斯（Cyclades/Naxos）

| 维度 | 详情 |
|------|------|
| **获取** | 1450年代——帝国海军以"清剿海盗、恢复帝国税法"为由，重炮轰开纳克索斯城墙，废黜威尼斯人建立的"纳克索斯公国"，改组为皇家直辖**"爱琴海军区"** |
| **1571年** | 距君堡极近，被西班牙舰队洗劫；1580年代即被流亡政权收复 |
| **现代** | 高端旅游业与古典考古学核心地带 |

### 十二群岛/罗德岛（Dodecanese/Rhodes）

| 维度 | 详情 |
|------|------|
| **获取** | 1510年——发动**"反向十字军"**，动用当时欧洲最庞大攻城炮兵，历经六个月围城战迫使医院骑士团（圣约翰骑士团）投降并驱逐至马耳他 |
| **现代** | 监视安纳托利亚南部海岸线与黎凡特北部的**雷达前哨与空军基地** |

## 五、已失殖民地（一战/巴黎和会剥夺）

在19世纪末"瓜分非洲"狂潮中，罗马共和国以亚历山大港为跳板进行短暂殖民扩张。1916年底罗马倒戈协约国后，英法在1919年巴黎和会上以"惩罚倒戈"为名彻底剥夺这些殖民地。

### 北非：昔兰尼加与的黎波里塔尼亚（今利比亚）

| 阶段 | 事件 |
|------|------|
| **1880s扩张** | 以"保护古罗马遗迹与基督教少数派"为由出兵占领；打造为中地中海海军堡垒，与法国突尼斯接壤 |
| **1919年丢失** | 英法煽动塞努西教团阿拉伯部落起义；操纵国际联盟将北非殖民地"委任"给**意大利与法国**瓜分——罗马眼睁睁看着宿敌意大利在北非站稳脚跟 |

### 东非：厄立特里亚沿海与索马里兰边缘

| 阶段 | 事件 |
|------|------|
| **1890s扩张** | 以"寻找古老基督教兄弟——祭司王约翰领地"为神圣借口大举南下；避免了现实中意大利在阿杜瓦的惨败，但陷入东非游击战泥潭 |
| **1919年丢失** | 英国以"保障苏伊士运河红海侧翼绝对安全"为由强行接管，并入英属东非或意属厄立特里亚——罗马彻底失去红海沿岸立足点 |

### 远东："新查士丁港"（Neo-Justinianopolis）

| 阶段 | 事件 |
|------|------|
| **1890s强租** | 八国联军侵华期间以"保护东正教传教士与远东贸易"为由，在华南沿海或南洋边缘强租微型加煤站与通商口岸 |
| **经济价值** | 几乎为零——驻扎一个连海军陆战队和一家东正教教堂，纯粹是"列强瓜分世界地图"上凑齐一块紫色拼图的政治面子工程 |
| **1914年沦陷** | 一战爆发第一周即被**英国皇家海军（或日本海军）炮轰并占领** |
| **1919年无视** | 巴黎和会上英法日以"战时敌产已合法没收"为由将其转交中国政府或英国海峡殖民地 |

**历史定性**：远东殖民地的丢失标志着罗马共和国彻底放弃"全球海洋帝国"幻梦。君堡政客痛苦意识到：**没有强大深蓝舰队，跨越半个地球的殖民地不过是列强案板上的鱼肉。** 此后战略重心全面收缩回东地中海、黑海海峡与中东。

## 六、海疆变迁的地缘逻辑

| 阶段 | 特征 |
|------|------|
| **上升期（1400–1500）** | 古典法统 + 复兴海军 → 肃清爱琴海拉丁人残余 → 东地中海内湖霸权 |
| **殖民巅峰（1880–1910）** | 帝国主义狂飙 → 北非、东非、远东 → 重温"环地中海-全球帝国"旧梦 |
| **收缩期（1919–至今）** | 巴黎和会背刺 → 剥离所有无法防守的远洋殖民地 → 全部资源倾注爱琴海岛屿链、亚历山大港飞地、苏伊士运河博弈 |

**核心教训**：克里特的潜艇还在潜航，罗德岛的雷达还在旋转，亚历山大港的灯塔还在闪烁——罗马不再追求"日不落"荣光，而是做东地中海不可撼动的海权霸主。
