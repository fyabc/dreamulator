# 末日预言与王朝更迭：东亚架空历史分支

## 核心前提

**圣历999年除夕夜，一场"注定降临"的世界末日并未发生。** 在此之前，一个起源于历法误差的科学警告，经过数百年的层累叠加与传播异变，被扭曲为不可逆转的灭世谶语，导致了整个东亚文明圈的崩溃与重塑。

核心基调：**黑色幽默**。毁灭人类的从来不是超自然的末日，而是人类在面对复杂科学问题时，因政治傲慢与群体无知所引发的认知灾难。

## 关键转折点

- **圣历~124年** — 历算大师"落下氏"发现上元历将在千年后达到数学奇点，留下科学警告
- **圣历~500年** — "历法误差"模因在民间异化为"劫火焚天"灭世预言
- **圣历~750年** — 朝廷颁布《禁私习天文图谶令》，预言转入地下并定型为谶语
- **圣历759年** — 大胤朝建立，收编预言为合法性来源，自诩"代天守岁"
- **圣历926年** — 裴肃改革失败，落雁谷沙尘暴"应验"伪谶语，天算师南渡（风劫）
- **圣历981年** — "盲日食"突然降临，上元历彻底破产
- **圣历982年** — 秘密地宫塌方暴露，胤朝合法性崩塌
- **圣历985年** — 陇右大地震，澜江上游堰塞成湖（地劫）
- **圣历990年** — 祈朝焚烧陪都引发次生大火（火劫）
- **圣历993年** — 堰塞湖溃坝，大洪水横扫帝国核心区（水劫）
- **圣历995年** — 《四劫坏空歌》席卷中原
- **圣历999年除夕** — 末日未至。太阳照常升起
- **圣历1001年** — 大启朝建立，废除上元历，确立极端实用主义
- **圣历1018年** — 《崇实诏》追尊裴肃，《四劫考误》颁布
- **圣历1307年** — 启朝死于数据造假与"新谶纬"反噬
- **圣历1334年** — 东北渊朝（温敦氏）入关，建立北朝
- **圣历1489年** — 大雍朝建立，"以心性驾驭算尺"
